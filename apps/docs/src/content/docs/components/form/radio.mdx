The **Radio** component provides a way to select a single option from a group of choices.

**Radio** là component cho phép chọn một tùy chọn duy nhất từ một nhóm các lựa chọn.

## Features | Tính Năng

- **Single selection** | **Chọn đơn**: Only one option can be selected at a time
- **Angular Forms integration** | **Tích hợp Angular Forms**: Works with ngModel and Reactive Forms
- **6 status colors** | **6 màu trạng thái**: Basic, primary, success, info, warning, danger
- **Disabled state** | **Trạng thái vô hiệu hóa**: Individual or group disable
- **Dark mode** | **Chế độ tối**: Full dark mode support
- **Keyboard navigation** | **Điều hướng bàn phím**: Arrow keys for selection

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { RadioComponent } from '@/components/ui/radio';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [RadioComponent],
  template: `
    <g-radio
      name="option"
      [options]="options"
      [(value)]="selectedOption">
    </g-radio>

    <p>Selected: {{ selectedOption() }}</p>
  `
})
export class ExampleComponent {
  selectedOption = signal('option1');

  options = [
    { label: 'Option 1', value: 'option1' },
    { label: 'Option 2', value: 'option2' },
    { label: 'Option 3', value: 'option3' }
  ];
}
```

### Reactive Forms | Forms Phản Ứng

```typescript
import { FormBuilder, FormGroup, ReactiveFormsModule } from '@angular/forms';

@Component({
  imports: [RadioComponent, ReactiveFormsModule],
  template: `
    <form [formGroup]="form">
      <g-radio
        name="plan"
        formControlName="selectedPlan"
        [options]="planOptions">
      </g-radio>
    </form>

    <p>Selected plan: {{ form.value.selectedPlan }}</p>
  `
})
export class PlanSelectorComponent {
  form: FormGroup;

  planOptions = [
    { label: 'Free', value: 'free' },
    { label: 'Pro', value: 'pro' },
    { label: 'Enterprise', value: 'enterprise' }
  ];

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      selectedPlan: ['pro']
    });
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `name` | `string` | required | Radio group name <br/> Tên nhóm radio |
| `options` | `RadioOption[]` | `[]` | Array of radio options <br/> Mảng các tùy chọn radio |
| `value` | `any` | - | Selected value <br/> Giá trị được chọn |
| `status` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'primary'` | Color theme <br/> Màu chủ đề |
| `disabled` | `boolean` | `false` | Disable entire group <br/> Vô hiệu hóa cả nhóm |
| `direction` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction <br/> Hướng bố cục |

### RadioOption Interface

```typescript
interface RadioOption {
  label: string;      // Display label | Nhãn hiển thị
  value: any;         // Option value | Giá trị tùy chọn
  disabled?: boolean; // Disable this option | Vô hiệu hóa tùy chọn này
}
```

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `valueChange` | `any` | Emitted when selection changes <br/> Phát ra khi lựa chọn thay đổi |

## Best Practices | Thực Hành Tốt Nhất

1. **Use for single selection** | **Dùng cho chọn đơn**: Use radio for mutually exclusive options

2. **Provide clear labels** | **Cung cấp nhãn rõ ràng**: Make option labels descriptive and concise

3. **Limit options** | **Giới hạn tùy chọn**: Keep to 5-7 options maximum, use Select for more

4. **Default selection** | **Lựa chọn mặc định**: Pre-select a sensible default option

5. **Group related options** | **Nhóm tùy chọn liên quan**: Use vertical layout for better readability
