The **Calendar Range** component provides an intuitive way to select a date range with start and end dates.

**Calendar Range** là component cung cấp cách trực quan để chọn khoảng ngày với ngày bắt đầu và kết thúc.

## Features | Tính Năng

- **Range selection** | **Chọn khoảng**: Select start and end dates
- **Min/max dates** | **Ngày tối thiểu/tối đa**: Restrict selectable date range
- **Quick presets** | **Cài đặt nhanh**: Common date range shortcuts
- **6 status colors** | **6 màu trạng thái**: Basic, primary, success, info, warning, danger
- **Angular Forms integration** | **Tích hợp Angular Forms**: Works with ngModel and Reactive Forms
- **Dark mode** | **Chế độ tối**: Full dark mode support
- **Localization** | **Bản địa hóa**: Support for multiple languages

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { CalendarRangeComponent } from '@/components/ui/calendar-range';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [CalendarRangeComponent],
  template: `
    <g-calendar-range
      [(startDate)]="startDate"
      [(endDate)]="endDate">
    </g-calendar-range>

    <p>
      Selected: {{ startDate() | date }} - {{ endDate() | date }}
    </p>
  `
})
export class ExampleComponent {
  startDate = signal<Date | null>(null);
  endDate = signal<Date | null>(null);
}
```

### With Custom Format | Với Định Dạng Tùy Chỉnh

```typescript
@Component({
  template: `
    <div class="space-y-4">
      <g-calendar-range
        [(startDate)]="startDate"
        [(endDate)]="endDate"
        [dateFormat]="dateFormat">
      </g-calendar-range>

      <div class="p-4 bg-gray-100 dark:bg-gray-800 rounded-lg">
        <p class="text-sm">
          Formatted: {{ formatDateRange() }}
        </p>
      </div>
    </div>
  `
})
export class CustomFormatComponent {
  startDate = signal<Date | null>(null);
  endDate = signal<Date | null>(null);
  dateFormat = 'dd/MM/yyyy';

  formatDateRange(): string {
    const start = this.startDate();
    const end = this.endDate();
    if (!start || !end) return 'No dates selected';
    return `${start.toLocaleDateString()} - ${end.toLocaleDateString()}`;
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `startDate` | `Date \| null` | `null` | Start date of range <br/> Ngày bắt đầu của khoảng |
| `endDate` | `Date \| null` | `null` | End date of range <br/> Ngày kết thúc của khoảng |
| `minDate` | `Date \| null` | `null` | Minimum selectable date <br/> Ngày tối thiểu có thể chọn |
| `maxDate` | `Date \| null` | `null` | Maximum selectable date <br/> Ngày tối đa có thể chọn |
| `status` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'primary'` | Color theme <br/> Màu chủ đề |
| `disabled` | `boolean` | `false` | Disable calendar <br/> Vô hiệu hóa lịch |
| `presets` | `DateRangePreset[]` | `[]` | Quick date range presets <br/> Cài đặt nhanh khoảng ngày |
| `dateFormat` | `string` | `'MM/dd/yyyy'` | Date display format <br/> Định dạng hiển thị ngày |

### DateRangePreset Interface

```typescript
interface DateRangePreset {
  label: string;                           // Display label | Nhãn hiển thị
  value: () => { start: Date; end: Date }; // Date range calculator | Tính khoảng ngày
}
```

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `startDateChange` | `Date \| null` | Emitted when start date changes <br/> Phát ra khi ngày bắt đầu thay đổi |
| `endDateChange` | `Date \| null` | Emitted when end date changes <br/> Phát ra khi ngày kết thúc thay đổi |
| `rangeChange` | `{ start: Date \| null, end: Date \| null }` | Emitted when range changes <br/> Phát ra khi khoảng thay đổi |

### Hotel Booking | Đặt Phòng Khách Sạn

```typescript
@Component({
  template: `
    <div class="bg-white dark:bg-gray-800 rounded-lg p-6 max-w-2xl space-y-6">
      <h3 class="text-lg font-semibold">
        Book Your Stay | Đặt Phòng
      </h3>

      <div class="grid grid-cols-1 md:grid-cols-2 gap-4">
        <div>
          <label class="block text-sm font-medium mb-2">
            Check-in / Check-out | Nhận / Trả Phòng
          </label>
          <g-calendar-range
            status="primary"
            [(startDate)]="checkIn"
            [(endDate)]="checkOut"
            [minDate]="today"
            [presets]="bookingPresets">
          </g-calendar-range>
        </div>

        <div class="space-y-4">
          <div>
            <label class="block text-sm font-medium mb-2">
              Guests | Khách
            </label>
            <input
              type="number"
              [(ngModel)]="guests"
              min="1"
              max="10"
              class="w-full px-3 py-2 border rounded-lg dark:bg-gray-700 dark:border-gray-600">
          </div>

          <div>
            <label class="block text-sm font-medium mb-2">
              Rooms | Phòng
            </label>
            <input
              type="number"
              [(ngModel)]="rooms"
              min="1"
              max="5"
              class="w-full px-3 py-2 border rounded-lg dark:bg-gray-700 dark:border-gray-600">
          </div>
        </div>
      </div>

      @if (checkIn() && checkOut()) {
        <div class="p-4 bg-blue-50 dark:bg-blue-900/20 rounded-lg">
          <h4 class="font-medium text-blue-900 dark:text-blue-100 mb-2">
            Booking Summary | Tóm Tắt Đặt Phòng
          </h4>
          <div class="text-sm text-blue-800 dark:text-blue-200 space-y-1">
            <p>Duration: {{ calculateNights() }} nights | {{ calculateNights() }} đêm</p>
            <p>Guests: {{ guests }} | Khách: {{ guests }}</p>
            <p>Rooms: {{ rooms }} | Phòng: {{ rooms }}</p>
            <p class="font-medium pt-2">Total: ${{ calculateTotal() }}</p>
          </div>
        </div>
      }

      <button
        class="w-full px-4 py-2 bg-primary-600 text-white rounded-lg hover:bg-primary-700 disabled:opacity-50"
        [disabled]="!checkIn() || !checkOut()"
        (click)="confirmBooking()">
        Confirm Booking | Xác Nhận Đặt Phòng
      </button>
    </div>
  `
})
export class HotelBookingComponent {
  checkIn = signal<Date | null>(null);
  checkOut = signal<Date | null>(null);
  guests = 2;
  rooms = 1;
  today = new Date();

  bookingPresets = [
    {
      label: 'Weekend | Cuối tuần',
      value: () => {
        const now = new Date();
        const dayOfWeek = now.getDay();
        const daysUntilFriday = (5 - dayOfWeek + 7) % 7;
        const friday = new Date(now.getTime() + daysUntilFriday * 24 * 60 * 60 * 1000);
        const sunday = new Date(friday.getTime() + 2 * 24 * 60 * 60 * 1000);
        return { start: friday, end: sunday };
      }
    },
    {
      label: '1 week | 1 tuần',
      value: () => ({
        start: new Date(),
        end: new Date(Date.now() + 7 * 24 * 60 * 60 * 1000)
      })
    },
    {
      label: '2 weeks | 2 tuần',
      value: () => ({
        start: new Date(),
        end: new Date(Date.now() + 14 * 24 * 60 * 60 * 1000)
      })
    }
  ];

  calculateNights(): number {
    const start = this.checkIn();
    const end = this.checkOut();
    if (!start || !end) return 0;
    const diff = end.getTime() - start.getTime();
    return Math.ceil(diff / (1000 * 60 * 60 * 24));
  }

  calculateTotal(): number {
    const pricePerNight = 100;
    return this.calculateNights() * this.rooms * pricePerNight;
  }

  confirmBooking() {
    console.log('Booking confirmed:', {
      checkIn: this.checkIn(),
      checkOut: this.checkOut(),
      guests: this.guests,
      rooms: this.rooms,
      total: this.calculateTotal()
    });
  }
}
```

### Event Calendar Filter | Bộ Lọc Lịch Sự Kiện

```typescript
@Component({
  template: `
    <div class="space-y-4">
      <div class="flex items-center gap-4">
        <g-calendar-range
          status="success"
          size="sm"
          [(startDate)]="filterStart"
          [(endDate)]="filterEnd"
          [presets]="eventPresets"
          (rangeChange)="onDateRangeChange($event)">
        </g-calendar-range>

        <button
          class="px-4 py-2 text-sm border rounded-lg hover:bg-gray-50 dark:hover:bg-gray-700"
          (click)="clearFilter()">
          Clear | Xóa
        </button>
      </div>

      <div class="space-y-2">
        @for (event of filteredEvents(); track event.id) {
          <div class="p-4 border rounded-lg dark:border-gray-700">
            <div class="flex items-start justify-between">
              <div>
                <h4 class="font-medium">{{ event.title }}</h4>
                <p class="text-sm text-gray-600 dark:text-gray-400">
                  {{ event.date | date:'medium' }}
                </p>
              </div>
              <span [class]="getEventBadgeClass(event.type)">
                {{ event.type }}
              </span>
            </div>
          </div>
        }

        @if (filteredEvents().length === 0) {
          <div class="text-center py-8 text-gray-500">
            No events found | Không tìm thấy sự kiện
          </div>
        }
      </div>
    </div>
  `
})
export class EventCalendarComponent {
  filterStart = signal<Date | null>(null);
  filterEnd = signal<Date | null>(null);

  events = signal([
    { id: 1, title: 'Team Meeting', date: new Date('2025-11-01'), type: 'meeting' },
    { id: 2, title: 'Product Launch', date: new Date('2025-11-05'), type: 'launch' },
    { id: 3, title: 'Conference', date: new Date('2025-11-10'), type: 'conference' },
    { id: 4, title: 'Training', date: new Date('2025-11-15'), type: 'training' }
  ]);

  filteredEvents = computed(() => {
    const start = this.filterStart();
    const end = this.filterEnd();

    if (!start || !end) return this.events();

    return this.events().filter(event =>
      event.date >= start && event.date <= end
    );
  });

  eventPresets = [
    {
      label: 'This week | Tuần này',
      value: () => {
        const now = new Date();
        const dayOfWeek = now.getDay();
        const start = new Date(now.getTime() - dayOfWeek * 24 * 60 * 60 * 1000);
        const end = new Date(start.getTime() + 6 * 24 * 60 * 60 * 1000);
        return { start, end };
      }
    },
    {
      label: 'Next 2 weeks | 2 tuần tới',
      value: () => ({
        start: new Date(),
        end: new Date(Date.now() + 14 * 24 * 60 * 60 * 1000)
      })
    },
    {
      label: 'This month | Tháng này',
      value: () => {
        const now = new Date();
        return {
          start: new Date(now.getFullYear(), now.getMonth(), 1),
          end: new Date(now.getFullYear(), now.getMonth() + 1, 0)
        };
      }
    }
  ];

  onDateRangeChange(range: { start: Date | null, end: Date | null }) {
    console.log('Date range changed:', range);
  }

  clearFilter() {
    this.filterStart.set(null);
    this.filterEnd.set(null);
  }

  getEventBadgeClass(type: string): string {
    const baseClass = 'px-2 py-1 text-xs rounded-full';
    const typeClasses: Record<string, string> = {
      meeting: 'bg-blue-100 text-blue-800 dark:bg-blue-900 dark:text-blue-200',
      launch: 'bg-green-100 text-green-800 dark:bg-green-900 dark:text-green-200',
      conference: 'bg-purple-100 text-purple-800 dark:bg-purple-900 dark:text-purple-200',
      training: 'bg-orange-100 text-orange-800 dark:bg-orange-900 dark:text-orange-200'
    };
    return `${baseClass} ${typeClasses[type] || ''}`;
  }
}
```

## Best Practices | Thực Hành Tốt Nhất

1. **Provide presets** | **Cung cấp cài đặt nhanh**: Add common date range shortcuts for convenience

2. **Set sensible limits** | **Đặt giới hạn hợp lý**: Use minDate/maxDate to prevent invalid selections

3. **Clear visual feedback** | **Phản hồi trực quan rõ ràng**: Highlight selected range distinctly

4. **Validate range** | **Xác thực khoảng**: Ensure end date is after start date

5. **Format dates appropriately** | **Định dạng ngày phù hợp**: Use locale-appropriate date formats

6. **Handle timezone** | **Xử lý múi giờ**: Be aware of timezone issues in date handling
