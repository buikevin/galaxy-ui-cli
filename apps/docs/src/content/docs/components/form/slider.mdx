The **Slider** component provides an intuitive way to select a numeric value from a range with a draggable handle.

**Slider** là component cung cấp cách trực quan để chọn giá trị số từ một khoảng với tay cầm kéo được.

## Features | Tính Năng

- **Range selection** | **Chọn khoảng**: Set min/max values with step control
- **Value display** | **Hiển thị giá trị**: Optional value label
- **6 status colors** | **6 màu trạng thái**: Basic, primary, success, info, warning, danger
- **Angular Forms integration** | **Tích hợp Angular Forms**: Works with ngModel and Reactive Forms
- **Disabled state** | **Trạng thái vô hiệu hóa**: Prevent user interaction
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { SliderComponent } from '@/components/ui/slider';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [SliderComponent],
  template: `
    <g-slider
      [min]="0"
      [max]="100"
      [(value)]="volume">
    </g-slider>

    <p>Volume: {{ volume() }}%</p>
  `
})
export class ExampleComponent {
  volume = signal(50);
}
```

### Reactive Forms | Forms Phản Ứng

```typescript
import { FormBuilder, FormGroup, ReactiveFormsModule } from '@angular/forms';

@Component({
  imports: [SliderComponent, ReactiveFormsModule],
  template: `
    <form [formGroup]="form">
      <div class="space-y-4">
        <div>
          <label class="block text-sm font-medium mb-2">
            Brightness | Độ Sáng
          </label>
          <g-slider
            formControlName="brightness"
            [min]="0"
            [max]="100"
            [showValue]="true">
          </g-slider>
        </div>

        <div>
          <label class="block text-sm font-medium mb-2">
            Contrast | Độ Tương Phản
          </label>
          <g-slider
            formControlName="contrast"
            [min]="0"
            [max]="100"
            [showValue]="true">
          </g-slider>
        </div>
      </div>
    </form>

    <pre>{{ form.value | json }}</pre>
  `
})
export class ImageAdjustmentComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      brightness: [50],
      contrast: [50]
    });
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `value` | `number` | `0` | Current value <br/> Giá trị hiện tại |
| `min` | `number` | `0` | Minimum value <br/> Giá trị tối thiểu |
| `max` | `number` | `100` | Maximum value <br/> Giá trị tối đa |
| `step` | `number` | `1` | Step increment <br/> Bước nhảy |
| `status` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'primary'` | Color theme <br/> Màu chủ đề |
| `disabled` | `boolean` | `false` | Disable slider <br/> Vô hiệu hóa thanh trượt |
| `showValue` | `boolean` | `false` | Show value label <br/> Hiển thị nhãn giá trị |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `valueChange` | `number` | Emitted when value changes <br/> Phát ra khi giá trị thay đổi |

## Accessibility | Khả Năng Tiếp Cận

- Uses semantic `<input type="range">` element
- Proper ARIA attributes (aria-valuemin, aria-valuemax, aria-valuenow)
- Keyboard accessible (Arrow keys to adjust)
- Focus visible indicators
- Screen reader announcements for value changes

          <label class="block text-sm font-medium mb-2">

        </div>

        <div class="flex gap-2 pt-4">

            (click)="reset()">
            Reset | Đặt Lại
          </button>
          <button
            class="flex-1 px-4 py-2 bg-primary-600 text-white rounded-lg hover:bg-primary-700"
            (click)="apply()">
            Apply | Áp Dụng
          </button>
        </div>
      </div>
    </div>
  `
})
export class ImageEditorComponent {
  brightness = signal(100);
  contrast = signal(100);
  saturation = signal(100);
  opacity = signal(1);

  reset() {
    this.brightness.set(100);
    this.contrast.set(100);
    this.saturation.set(100);
    this.opacity.set(1);
  }

  apply() {
    console.log('Apply adjustments:', {
      brightness: this.brightness(),
      contrast: this.contrast(),
      saturation: this.saturation(),
      opacity: this.opacity()
    });
  }
}

## Best Practices | Thực Hành Tốt Nhất

1. **Show value feedback** | **Hiển thị phản hồi giá trị**: Enable showValue or display custom value label

2. **Appropriate range** | **Khoảng phù hợp**: Set min/max values that make sense for the context

3. **Step increments** | **Bước nhảy**: Use appropriate step values for precision needed

4. **Visual feedback** | **Phản hồi trực quan**: Use status colors to indicate importance or warning levels

5. **Labels and context** | **Nhãn và ngữ cảnh**: Always provide labels describing what the slider controls

6. **Unit display** | **Hiển thị đơn vị**: Include units in labels (%, px, dB, etc.)
