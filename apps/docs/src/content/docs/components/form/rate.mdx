The **Rate** component provides an intuitive way to collect star ratings from users with customizable features.

**Rate** là component cung cấp cách trực quan để thu thập đánh giá sao từ người dùng với các tính năng tùy chỉnh.

## Features | Tính Năng

- **Customizable stars** | **Tùy chỉnh sao**: Set custom star count (default 5)
- **Half star support** | **Hỗ trợ nửa sao**: Allow half-star ratings
- **Read-only mode** | **Chế độ chỉ đọc**: Display ratings without editing
- **6 status colors** | **6 màu trạng thái**: Basic, primary, success, info, warning, danger
- **Angular Forms integration** | **Tích hợp Angular Forms**: Works with ngModel and Reactive Forms
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { RateComponent } from '@/components/ui/rate';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [RateComponent],
  template: `
    <g-rate [(value)]="rating">
    </g-rate>

    <p>Rating: {{ rating() }} stars</p>
  `
})
export class ExampleComponent {
  rating = signal(0);
}
```

### Reactive Forms | Forms Phản Ứng

```typescript
import { FormBuilder, FormGroup, ReactiveFormsModule } from '@angular/forms';
import { Validators } from '@angular/forms';

@Component({
  imports: [RateComponent, ReactiveFormsModule],
  template: `
    <form [formGroup]="form" (ngSubmit)="submitReview()">
      <div class="space-y-4">
        <div>
          <label class="block text-sm font-medium mb-2">
            Overall Rating | Đánh Giá Chung
          </label>
          <g-rate
            formControlName="rating"
            [allowHalf]="true">
          </g-rate>
          @if (form.get('rating')?.hasError('required') && form.get('rating')?.touched) {
            <p class="text-sm text-red-600 mt-1">
              Rating is required | Đánh giá là bắt buộc
            </p>
          }
        </div>

        <button
          type="submit"
          class="px-4 py-2 bg-primary-600 text-white rounded-lg hover:bg-primary-700"
          [disabled]="form.invalid">
          Submit Review | Gửi Đánh Giá
        </button>
      </div>
    </form>
  `
})
export class ReviewFormComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      rating: [0, [Validators.required, Validators.min(1)]]
    });
  }

  submitReview() {
    if (this.form.valid) {
      console.log('Review submitted:', this.form.value);
    }
  }
}
```

### With Text Description | Với Mô Tả Văn Bản

```typescript
@Component({
  template: `
    <div class="space-y-2">
      <g-rate
        [(value)]="rating"
        (valueChange)="onRatingChange($event)">
      </g-rate>
      <p class="text-sm text-gray-600 dark:text-gray-400">
        {{ getRatingText() }}
      </p>
    </div>
  `
})
export class RateWithTextComponent {
  rating = signal(0);

  ratingDescriptions = [
    'Select a rating | Chọn đánh giá',
    'Poor | Kém',
    'Fair | Trung bình',
    'Good | Tốt',
    'Very Good | Rất tốt',
    'Excellent | Xuất sắc'
  ];

  getRatingText(): string {
    return this.ratingDescriptions[this.rating()] || this.ratingDescriptions[0];
  }

  onRatingChange(value: number) {
    console.log('Rating changed to:', value);
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `value` | `number` | `0` | Current rating value <br/> Giá trị đánh giá hiện tại |
| `count` | `number` | `5` | Total number of stars <br/> Tổng số sao |
| `allowHalf` | `boolean` | `false` | Allow half-star ratings <br/> Cho phép đánh giá nửa sao |
| `readonly` | `boolean` | `false` | Read-only mode <br/> Chế độ chỉ đọc |
| `status` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'warning'` | Color theme <br/> Màu chủ đề |
| `disabled` | `boolean` | `false` | Disable rating <br/> Vô hiệu hóa đánh giá |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `valueChange` | `number` | Emitted when rating changes <br/> Phát ra khi đánh giá thay đổi |

### Feedback Widget | Widget Phản Hồi

```typescript
@Component({
  template: `
    <div class="bg-white dark:bg-gray-800 rounded-lg p-6 max-w-md mx-auto">
      <h3 class="text-lg font-semibold mb-4 text-center">
        How was your experience? | Trải nghiệm của bạn thế nào?
      </h3>

      <div class="flex flex-col items-center space-y-4">
        <g-rate
          status="warning"
          [count]="5"
          [(value)]="rating"
          (valueChange)="onRatingChange($event)">
        </g-rate>

        @if (rating() > 0) {
          <div class="text-center">
            <p class="text-2xl mb-2">{{ getEmoji() }}</p>
            <p class="text-sm text-gray-600 dark:text-gray-400">
              {{ getFeedbackText() }}
            </p>
          </div>
        }

        @if (rating() > 0) {
          <textarea
            [(ngModel)]="feedback"
            rows="3"
            class="w-full px-3 py-2 border rounded-lg dark:bg-gray-700 dark:border-gray-600"
            placeholder="Tell us more... | Cho chúng tôi biết thêm...">
          </textarea>

          <button
            class="w-full px-4 py-2 bg-primary-600 text-white rounded-lg hover:bg-primary-700"
            (click)="submitFeedback()">
            Submit Feedback | Gửi Phản Hồi
          </button>
        }
      </div>
    </div>
  `
})
export class FeedbackWidgetComponent {
  rating = signal(0);
  feedback = '';

  feedbackData = [
    { emoji: '', text: '' },
    { emoji: '😢', text: 'We\'re sorry | Chúng tôi xin lỗi' },
    { emoji: '😕', text: 'Could be better | Có thể tốt hơn' },
    { emoji: '😊', text: 'Good | Tốt' },
    { emoji: '😃', text: 'Great | Tuyệt vời' },
    { emoji: '🤩', text: 'Amazing! | Tuyệt vời!' }
  ];

  getEmoji(): string {
    return this.feedbackData[this.rating()]?.emoji || '';
  }

  getFeedbackText(): string {
    return this.feedbackData[this.rating()]?.text || '';
  }

  onRatingChange(value: number) {
    console.log('Rating:', value);
  }

  submitFeedback() {
    console.log('Feedback submitted:', {
      rating: this.rating(),
      feedback: this.feedback
    });
  }
}
```

## Best Practices | Thực Hành Tốt Nhất

1. **Use appropriate scale** | **Dùng thang đo phù hợp**: 5 stars is standard, use others sparingly

2. **Allow half stars for precision** | **Cho phép nửa sao cho chính xác**: Enable allowHalf for more granular ratings

3. **Provide context** | **Cung cấp ngữ cảnh**: Add labels or text to explain what's being rated

4. **Visual feedback** | **Phản hồi trực quan**: Show rating text or emoji for better UX

5. **Read-only for display** | **Chỉ đọc để hiển thị**: Use readonly mode when showing existing ratings

6. **Validation** | **Xác thực**: Require ratings in forms where applicable
