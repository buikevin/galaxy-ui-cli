The **Popover** component displays rich content in a floating panel, triggered by user interaction. Inspired by Ant Design's Popover.

**Popover** là component hiển thị nội dung phong phú trong một panel nổi, được kích hoạt bởi tương tác người dùng. Lấy cảm hứng từ Popover của Ant Design.

## Features | Tính Năng

- **12 placement options** | **12 vị trí đặt**: Top, bottom, left, right with start/end variants
- **3 trigger types** | **3 kiểu kích hoạt**: Hover, click, focus
- **Custom content** | **Nội dung tùy chỉnh**: Support for templates and components
- **Title support** | **Hỗ trợ tiêu đề**: Optional header with title
- **Controlled visibility** | **Kiểm soát hiển thị**: Programmatic show/hide
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component } from '@angular/core';
import { PopoverComponent } from '@/components/ui/popover';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [PopoverComponent],
  template: `
    <g-popover
      [content]="'This is popover content'"
      [title]="'Popover Title'">
      <button>Hover me</button>
    </g-popover>
  `
})
export class ExampleComponent {}
```

### Trigger Types | Kiểu Kích Hoạt

```typescript
@Component({
  template: `
    <!-- Hover trigger (default) | Kích hoạt khi hover (mặc định) -->
    <g-popover
      trigger="hover"
      [content]="'Show on hover'">
      <button>Hover</button>
    </g-popover>

    <!-- Click trigger | Kích hoạt khi click -->
    <g-popover
      trigger="click"
      [content]="'Show on click'">
      <button>Click</button>
    </g-popover>

    <!-- Focus trigger | Kích hoạt khi focus -->
    <g-popover
      trigger="focus"
      [content]="'Show on focus'">
      <input type="text" placeholder="Focus me">
    </g-popover>
  `
})
```

### Controlled Visibility | Kiểm Soát Hiển Thị

```typescript
import { Component, signal } from '@angular/core';

@Component({
  template: `
    <div class="space-x-4">
      <g-popover
        [visible]="isVisible()"
        (visibleChange)="isVisible.set($event)"
        [content]="'Controlled popover content'">
        <button>Controlled Popover</button>
      </g-popover>

      <button (click)="isVisible.set(!isVisible())">
        {{ isVisible() ? 'Hide' : 'Show' }} Popover
      </button>
    </div>
  `
})
export class ControlledPopover {
  isVisible = signal(false);
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `content` | `string` | - | Popover content text <br/> Văn bản nội dung popover |
| `contentTemplate` | `TemplateRef` | - | Custom content template <br/> Template nội dung tùy chỉnh |
| `title` | `string` | - | Popover title <br/> Tiêu đề popover |
| `placement` | `'top' \| 'top-start' \| 'top-end' \| 'bottom' \| 'bottom-start' \| 'bottom-end' \| 'left' \| 'left-start' \| 'left-end' \| 'right' \| 'right-start' \| 'right-end'` | `'top'` | Popover placement <br/> Vị trí popover |
| `trigger` | `'hover' \| 'click' \| 'focus'` | `'hover'` | Trigger type <br/> Kiểu kích hoạt |
| `visible` | `boolean` | - | Control visibility <br/> Kiểm soát hiển thị |
| `disabled` | `boolean` | `false` | Disable popover <br/> Vô hiệu hóa popover |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `visibleChange` | `boolean` | Emitted when visibility changes <br/> Phát ra khi hiển thị thay đổi |

## Accessibility | Khả Năng Tiếp Cận

- Proper `aria-describedby` linking trigger to content
- Keyboard navigation support (Escape to close)
- Focus management when opened via click
- Screen reader announcements

## Best Practices | Thực Hành Tốt Nhất

1. **Keep content concise** | **Giữ nội dung ngắn gọn**: Popovers should provide supplementary information, not replace main content

2. **Appropriate trigger** | **Kích hoạt phù hợp**: Use hover for quick info, click for interactive content

3. **Placement awareness** | **Nhận biết vị trí**: Consider screen space and auto-adjust placement when needed

4. **Don't nest popovers** | **Không lồng popover**: Avoid opening popovers within popovers

5. **Mobile consideration** | **Xem xét mobile**: Use click trigger on mobile devices instead of hover
