The **Skeleton** component provides a placeholder preview of content before data loads. Inspired by Ant Design's Skeleton component with avatar, title, and paragraph options.

**Skeleton** là component cung cấp bản xem trước placeholder của nội dung trước khi dữ liệu tải. Lấy cảm hứng từ Skeleton component của Ant Design với các tùy chọn avatar, tiêu đề, và đoạn văn.

## Features | Tính Năng

- **Flexible layouts** | **Bố cục linh hoạt**: Avatar, title, paragraph combinations
- **Animated loading** | **Tải động**: Wave animation effect
- **Custom shapes** | **Hình dạng tùy chỉnh**: Circle, square, rounded variants
- **Responsive sizing** | **Kích thước đáp ứng**: Configurable width and height
- **Loading state control** | **Kiểm soát trạng thái tải**: Toggle between skeleton and content
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { SkeletonComponent } from '@/components/ui/skeleton';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [SkeletonComponent],
  template: `
    <g-skeleton [loading]="isLoading()">
      <div class="content">
        <h2>Content Title</h2>
        <p>Your actual content goes here...</p>
      </div>
    </g-skeleton>
  `
})
export class ExampleComponent {
  isLoading = signal(true);
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `loading` | `boolean` | `true` | Show skeleton or content <br/> Hiển thị skeleton hoặc nội dung |
| `active` | `boolean` | `true` | Show animation effect <br/> Hiển thị hiệu ứng hoạt ảnh |
| `avatar` | `boolean \| AvatarProps` | `false` | Show avatar placeholder <br/> Hiển thị placeholder avatar |
| `title` | `boolean \| TitleProps` | `true` | Show title placeholder <br/> Hiển thị placeholder tiêu đề |
| `paragraph` | `boolean \| ParagraphProps` | `true` | Show paragraph placeholder <br/> Hiển thị placeholder đoạn văn |
| `round` | `boolean` | `true` | Rounded corners <br/> Bo góc |

### AvatarProps Interface

```typescript
interface AvatarProps {
  shape?: 'circle' | 'square';  // Avatar shape | Hình dạng avatar
  size?: 'sm' | 'default' | 'lg' | number;  // Avatar size | Kích thước avatar
}
```

### TitleProps Interface

```typescript
interface TitleProps {
  width?: string | number;  // Title width | Chiều rộng tiêu đề
}
```

### ParagraphProps Interface

```typescript
interface ParagraphProps {
  rows?: number;  // Number of rows | Số lượng hàng
  width?: string | number | Array<string | number>;  // Row widths | Chiều rộng hàng
}
```

### List Loading | Tải Danh Sách

```typescript
@Component({
  template: `
    <div class="space-y-4">
      @if (isLoading()) {
        @for (i of [1, 2, 3, 4, 5]; track i) {
          <g-skeleton
            [loading]="true"
            [avatar]="{ size: 'lg', shape: 'circle' }"
            [paragraph]="{ rows: 2 }" />
        }
      } @else {
        @for (user of users(); track user.id) {
          <div class="flex gap-4 items-start">
            <img [src]="user.avatar" class="w-12 h-12 rounded-full" />
            <div class="flex-1">
              <h4 class="font-semibold">{{ user.name }}</h4>
              <p class="text-gray-600 text-sm">{{ user.email }}</p>
              <p class="text-gray-500 text-sm mt-1">{{ user.bio }}</p>
            </div>
          </div>
        }
      }
    </div>
  `
})
export class ListLoading {
  isLoading = signal(true);
  users = signal([]);

  constructor() {
    this.loadUsers();
  }

  async loadUsers() {
    this.isLoading.set(true);
    const data = await this.api.getUsers();
    this.users.set(data);
    this.isLoading.set(false);
  }
}
```

### Table Loading | Tải Bảng

```typescript
@Component({
  template: `
    <div class="border rounded-lg overflow-hidden">
      <table class="w-full">
        <thead class="bg-gray-50">
          <tr>
            <th class="px-6 py-3 text-left">Name</th>
            <th class="px-6 py-3 text-left">Email</th>
            <th class="px-6 py-3 text-left">Role</th>
            <th class="px-6 py-3 text-left">Status</th>
          </tr>
        </thead>
        <tbody>
          @if (isLoading()) {
            @for (i of [1, 2, 3, 4, 5]; track i) {
              <tr class="border-t">
                <td class="px-6 py-4">
                  <div class="h-4 bg-gray-200 rounded animate-pulse w-32"></div>
                </td>
                <td class="px-6 py-4">
                  <div class="h-4 bg-gray-200 rounded animate-pulse w-48"></div>
                </td>
                <td class="px-6 py-4">
                  <div class="h-4 bg-gray-200 rounded animate-pulse w-24"></div>
                </td>
                <td class="px-6 py-4">
                  <div class="h-4 bg-gray-200 rounded animate-pulse w-20"></div>
                </td>
              </tr>
            }
          } @else {
            @for (row of data(); track row.id) {
              <tr class="border-t">
                <td class="px-6 py-4">{{ row.name }}</td>
                <td class="px-6 py-4">{{ row.email }}</td>
                <td class="px-6 py-4">{{ row.role }}</td>
                <td class="px-6 py-4">
                  <span class="px-2 py-1 rounded text-sm" [class]="getStatusClass(row.status)">
                    {{ row.status }}
                  </span>
                </td>
              </tr>
            }
          }
        </tbody>
      </table>
    </div>
  `
})
export class TableLoading {
  isLoading = signal(true);
  data = signal([]);

  constructor() {
    this.loadData();
  }

  async loadData() {
    this.isLoading.set(true);
    const result = await this.api.getTableData();
    this.data.set(result);
    this.isLoading.set(false);
  }

  getStatusClass(status: string): string {
    const classes = {
      active: 'bg-green-100 text-green-800',
      inactive: 'bg-gray-100 text-gray-800',
      pending: 'bg-yellow-100 text-yellow-800'
    };
    return classes[status] || '';
  }
}
```

## Best Practices | Thực Hành Tốt Nhất

1. **Match content structure** | **Khớp cấu trúc nội dung**: Skeleton should mirror the actual content layout

2. **Use appropriate animation** | **Dùng hoạt ảnh phù hợp**: Enable animation for better UX, disable for performance

3. **Show realistic timing** | **Hiển thị thời gian thực tế**: Don't hide skeleton too quickly (min 300-500ms)

4. **Avoid flash** | **Tránh nhấp nháy**: If content loads very fast, don't show skeleton

5. **Progressive loading** | **Tải tiến bộ**: Show skeleton for individual items as they load
