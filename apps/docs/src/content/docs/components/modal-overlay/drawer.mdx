import { Tabs, TabItem } from '@astrojs/starlight/components';

The **Drawer** component provides a panel that slides in from the edge of the screen, perfect for navigation menus, filters, or forms. Inspired by shadcn's Sheet component.

**Drawer** là component cung cấp một panel trượt vào từ cạnh màn hình, hoàn hảo cho menu điều hướng, bộ lọc, hoặc form. Lấy cảm hứng từ Sheet component của shadcn.

## Features | Tính Năng

- **4 placements** | **4 vị trí**: Left, right, top, bottom
- **Body scroll lock** | **Khóa cuộn body**: Prevents background scrolling when open
- **Custom size** | **Kích thước tùy chỉnh**: Flexible width/height configuration
- **Backdrop overlay** | **Lớp phủ nền**: Semi-transparent backdrop with click-to-close
- **Keyboard support** | **Hỗ trợ bàn phím**: ESC key to close
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { DrawerComponent } from '@/components/ui/drawer';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [DrawerComponent],
  template: `
    <button (click)="isOpen.set(true)">Open Drawer</button>

    <g-drawer
      [open]="isOpen()"
      (openChange)="isOpen.set($event)"
      [title]="'Drawer Title'">
      <div class="p-4">
        <p>Drawer content goes here</p>
      </div>
    </g-drawer>
  `
})
export class ExampleComponent {
  isOpen = signal(false);
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `open` | `boolean` | `false` | Controls drawer visibility <br/> Kiểm soát hiển thị ngăn kéo |
| `placement` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'left'` | Drawer position <br/> Vị trí ngăn kéo |
| `title` | `string` | - | Drawer title <br/> Tiêu đề ngăn kéo |
| `size` | `'sm' \| 'md' \| 'lg' \| 'full'` | `'md'` | Predefined size <br/> Kích thước định sẵn |
| `customSize` | `string` | - | Custom width/height (e.g., '500px') <br/> Chiều rộng/cao tùy chỉnh |
| `closable` | `boolean` | `true` | Show close button <br/> Hiển thị nút đóng |
| `closeOnBackdropClick` | `boolean` | `true` | Close when clicking backdrop <br/> Đóng khi click vào nền |
| `closeOnEscape` | `boolean` | `true` | Close on ESC key <br/> Đóng khi nhấn ESC |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `openChange` | `boolean` | Emitted when drawer opens/closes <br/> Phát ra khi ngăn kéo mở/đóng |
| `afterOpen` | `void` | Emitted after drawer opened <br/> Phát ra sau khi ngăn kéo mở |
| `afterClose` | `void` | Emitted after drawer closed <br/> Phát ra sau khi ngăn kéo đóng |

## Accessibility | Khả Năng Tiếp Cận

- Uses ARIA `role="dialog"` and `aria-modal="true"`
- Proper focus management (traps focus within drawer)
- Keyboard navigation (ESC to close, Tab for focus cycling)
- Screen reader announcements for open/close states
- Prevents background scroll when open

## Best Practices | Thực Hành Tốt Nhất

1. **Appropriate placement** | **Vị trí phù hợp**: Use left/right for navigation and forms, top/bottom for filters or notifications

2. **Size consideration** | **Xem xét kích thước**: Don't make drawers too wide - leave some background visible

3. **Focus management** | **Quản lý focus**: Ensure focus moves to drawer when opened and returns to trigger when closed

4. **Mobile responsive** | **Đáp ứng mobile**: Consider full-width drawers on mobile devices

5. **Loading states** | **Trạng thái loading**: Show loading indicators when fetching drawer content
