The **Tour** component provides an elegant way to guide users through your application's features with step-by-step instructions and spotlight effects, inspired by Ant Design.

**Tour** là component thanh lịch để hướng dẫn người dùng qua các tính năng của ứng dụng với hướng dẫn từng bước và hiệu ứng làm nổi bật, lấy cảm hứng từ Ant Design.

## Features | Tính Năng

- **Spotlight effect** | **Hiệu ứng làm nổi bật**: Highlights target elements with mask overlay
- **Step-by-step navigation** | **Điều hướng từng bước**: Next/previous/skip controls
- **Multiple placements** | **Nhiều vị trí**: 12 placement options for popover positioning
- **Cover images** | **Ảnh bìa**: Optional images for each step
- **Mask overlay** | **Lớp phủ mờ**: Darkens background to focus attention
- **Auto scroll** | **Tự động cuộn**: Scrolls target elements into view
- **Keyboard support** | **Hỗ trợ bàn phím**: ESC to close, arrow keys for navigation
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { TourComponent, TourStep } from '@/components/ui/tour';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [TourComponent],
  template: `
    <!-- Elements to highlight | Phần tử để làm nổi bật -->
    <button id="step1" class="px-4 py-2 bg-blue-600 text-white rounded">
      Create New
    </button>

    <div id="step2" class="mt-4 p-4 border rounded">
      Your content here
    </div>

    <!-- Tour component | Component tour -->
    <g-tour
      [open]="tourOpen()"
      [steps]="tourSteps"
      [current]="currentStep()"
      (finish)="handleFinish()"
      (close)="handleClose()"
      (change)="handleChange($event)">
    </g-tour>

    <!-- Start tour button | Nút bắt đầu tour -->
    <button (click)="startTour()">Start Tour | Bắt Đầu Tour</button>
  `
})
export class ExampleComponent {
  tourOpen = signal(false);
  currentStep = signal(0);

  tourSteps: TourStep[] = [
    {
      target: '#step1',
      title: 'Create Button',
      description: 'Click here to create a new item',
      placement: 'bottom'
    },
    {
      target: '#step2',
      title: 'Content Area',
      description: 'Your content will appear here',
      placement: 'top'
    }
  ];

  startTour() {
    this.tourOpen.set(true);
    this.currentStep.set(0);
  }

  handleFinish() {
    console.log('Tour finished!');
    this.tourOpen.set(false);
  }

  handleClose() {
    this.tourOpen.set(false);
  }

  handleChange(step: number) {
    this.currentStep.set(step);
  }
}
```

### Controlled Mode | Chế Độ Có Kiểm Soát

Full control over tour state | Kiểm soát hoàn toàn trạng thái tour:

```typescript
@Component({
  template: `
    <g-tour
      [open]="isOpen()"
      [steps]="steps"
      [current]="currentStep()"
      (change)="onStepChange($event)"
      (close)="onClose()">
    </g-tour>

    <!-- Custom controls | Điều khiển tùy chỉnh -->
    <div class="flex gap-2">
      <button (click)="previousStep()">Previous | Trước</button>
      <button (click)="nextStep()">Next | Tiếp</button>
      <button (click)="skipTour()">Skip | Bỏ Qua</button>
    </div>
  `
})
export class ControlledTourComponent {
  isOpen = signal(true);
  currentStep = signal(0);

  steps: TourStep[] = [/* ... */];

  onStepChange(step: number) {
    this.currentStep.set(step);
  }

  onClose() {
    this.isOpen.set(false);
  }

  previousStep() {
    if (this.currentStep() > 0) {
      this.currentStep.update(step => step - 1);
    }
  }

  nextStep() {
    if (this.currentStep() < this.steps.length - 1) {
      this.currentStep.update(step => step + 1);
    }
  }

  skipTour() {
    this.isOpen.set(false);
  }
}
```

### Multi-Page Tour | Tour Nhiều Trang

Tour across different routes | Tour qua các route khác nhau:

```typescript
import { Router } from '@angular/router';

@Component({
  template: `
    <g-tour
      [open]="tourOpen()"
      [steps]="currentPageSteps()"
      [current]="currentStep()"
      (finish)="handleFinish()"
      (close)="handleClose()">
    </g-tour>
  `
})
export class MultiPageTourComponent {
  constructor(private router: Router) {}

  tourOpen = signal(false);
  currentStep = signal(0);
  currentPage = signal('dashboard');

  allSteps: Record<string, TourStep[]> = {
    dashboard: [
      { target: '#overview', title: 'Overview', description: '...' },
      { target: '#stats', title: 'Statistics', description: '...' }
    ],
    settings: [
      { target: '#profile', title: 'Profile', description: '...' },
      { target: '#security', title: 'Security', description: '...' }
    ]
  };

  currentPageSteps = computed(() => this.allSteps[this.currentPage()]);

  async handleFinish() {
    const currentPageSteps = this.currentPageSteps();

    if (this.currentStep() >= currentPageSteps.length - 1) {
      // Move to next page | Chuyển sang trang tiếp theo
      if (this.currentPage() === 'dashboard') {
        this.currentPage.set('settings');
        this.currentStep.set(0);
        await this.router.navigate(['/settings']);
      } else {
        // Tour complete | Tour hoàn thành
        this.tourOpen.set(false);
      }
    }
  }
}
```

## Props | Thuộc Tính

### TourComponent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `open` | `boolean` | `false` | Whether tour is visible <br/> Tour có hiển thị không |
| `steps` | `TourStep[]` | `[]` | Array of tour steps <br/> Mảng các bước tour |
| `current` | `number` | `0` | Current step index <br/> Chỉ số bước hiện tại |
| `mask` | `boolean` | `true` | Show mask overlay <br/> Hiển thị lớp phủ mờ |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `close` | `void` | Emitted when tour closes <br/> Phát ra khi tour đóng |
| `finish` | `void` | Emitted when tour completes <br/> Phát ra khi tour hoàn thành |
| `change` | `number` | Emitted when step changes <br/> Phát ra khi bước thay đổi |

### TourStep Interface

```typescript
interface TourStep {
  target: string;           // CSS selector | Bộ chọn CSS
  title: string;            // Step title | Tiêu đề bước
  description: string;      // Step description | Mô tả bước
  cover?: string;           // Optional cover image | Ảnh bìa tùy chọn
  placement?: TourPlacement; // Popover placement | Vị trí popover
}

type TourPlacement =
  | 'top' | 'top-start' | 'top-end'
  | 'bottom' | 'bottom-start' | 'bottom-end'
  | 'left' | 'left-start' | 'left-end'
  | 'right' | 'right-start' | 'right-end';
```

## Examples | Ví Dụ

### Onboarding Tour | Tour Giới Thiệu

```typescript
@Component({
  template: `
    <g-tour
      [open]="showOnboarding()"
      [steps]="onboardingSteps"
      [current]="onboardingStep()"
      (finish)="completeOnboarding()"
      (close)="skipOnboarding()">
    </g-tour>
  `
})
export class OnboardingComponent implements OnInit {
  showOnboarding = signal(false);
  onboardingStep = signal(0);

  onboardingSteps: TourStep[] = [
    {
      target: '#welcome',
      title: 'Welcome! | Chào Mừng!',
      description: 'Let me show you around our platform.',
      cover: '/images/welcome.jpg',
      placement: 'bottom'
    },
    {
      target: '#create-button',
      title: 'Create Your First Project | Tạo Dự Án Đầu Tiên',
      description: 'Click here to get started with your first project.',
      placement: 'right'
    },
    {
      target: '#templates',
      title: 'Browse Templates | Duyệt Mẫu',
      description: 'Choose from our pre-built templates to save time.',
      placement: 'left'
    },
    {
      target: '#help',
      title: 'Need Help? | Cần Giúp Đỡ?',
      description: 'Access documentation and support anytime.',
      placement: 'bottom'
    }
  ];

  ngOnInit() {
    // Check if user is new | Kiểm tra nếu người dùng mới
    const hasSeenOnboarding = localStorage.getItem('onboarding_complete');
    if (!hasSeenOnboarding) {
      this.showOnboarding.set(true);
    }
  }

  completeOnboarding() {
    localStorage.setItem('onboarding_complete', 'true');
    this.showOnboarding.set(false);
  }

  skipOnboarding() {
    this.showOnboarding.set(false);
  }
}
```

## Best Practices | Thực Hành Tốt Nhất

1. **Keep steps concise** | **Giữ các bước ngắn gọn**: 3-5 steps is ideal for user attention

2. **Use cover images** | **Sử dụng ảnh bìa**: Visual aids help users understand faster

3. **Allow skipping** | **Cho phép bỏ qua**: Always provide a skip/close option

4. **Save tour state** | **Lưu trạng thái tour**: Use localStorage to not show tour again

5. **Test placements** | **Kiểm tra vị trí**: Ensure popover is visible on all screen sizes

6. **Bilingual support** | **Hỗ trợ song ngữ**: Provide content in user's preferred language

7. **Feature-specific tours** | **Tour theo tính năng**: Create separate tours for different features

8. **Version tours** | **Tour theo phiên bản**: Show tours for new feature releases
