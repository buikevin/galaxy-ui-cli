import { Tabs, TabItem } from '@astrojs/starlight/components';

The **Carousel** component provides a smooth, animated way to cycle through images and content, inspired by Ant Design's carousel implementation.

**Carousel** là component mượt mà, có hoạt ảnh để xoay vòng qua ảnh và nội dung, lấy cảm hứng từ carousel của Ant Design.

## Features | Tính Năng

- **Autoplay** | **Tự động phát**: Automatic slide transitions with custom speed
- **Effects** | **Hiệu ứng**: Scroll and fade transition effects
- **Dots indicator** | **Chỉ thị chấm**: Navigation dots with active state
- **Navigation arrows** | **Mũi tên điều hướng**: Previous/next arrow buttons
- **Infinite loop** | **Vòng lặp vô hạn**: Seamless looping through slides
- **Touch support** | **Hỗ trợ chạm**: Swipe gestures on mobile devices
- **Keyboard navigation** | **Điều hướng bàn phím**: Arrow keys for slide control
- **Dark mode** | **Chế độ tối**: Full dark mode support

### Controlled Carousel | Carousel Có Kiểm Soát

```typescript
@Component({
  template: `
    <g-carousel
      [currentSlide]="activeSlide()"
      (afterChange)="onSlideChange($event)">
      <g-carousel-item>
        <img src="/slides/slide1.jpg" alt="Slide 1">
      </g-carousel-item>
      <g-carousel-item>
        <img src="/slides/slide2.jpg" alt="Slide 2">
      </g-carousel-item>
      <g-carousel-item>
        <img src="/slides/slide3.jpg" alt="Slide 3">
      </g-carousel-item>
    </g-carousel>

    <!-- Custom controls | Điều khiển tùy chỉnh -->
    <div class="flex justify-center gap-2 mt-4">
      <button (click)="goToSlide(0)">1</button>
      <button (click)="goToSlide(1)">2</button>
      <button (click)="goToSlide(2)">3</button>
    </div>
  `
})
export class ControlledCarouselComponent {
  activeSlide = signal(0);

  onSlideChange(index: number) {
    this.activeSlide.set(index);
    console.log('Slide changed to:', index);
  }

  goToSlide(index: number) {
    this.activeSlide.set(index);
  }
}
```

## Props | Thuộc Tính

### CarouselComponent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `autoplay` | `boolean` | `false` | Enable autoplay <br/> Bật tự động phát |
| `autoplaySpeed` | `number` | `3000` | Autoplay interval in ms <br/> Khoảng thời gian tự động phát (ms) |
| `arrows` | `boolean` | `false` | Show navigation arrows <br/> Hiển thị mũi tên điều hướng |
| `dots` | `boolean` | `true` | Show dot indicators <br/> Hiển thị chỉ thị chấm |
| `infinite` | `boolean` | `true` | Enable infinite loop <br/> Bật vòng lặp vô hạn |
| `effect` | `'scroll' \| 'fade'` | `'scroll'` | Transition effect <br/> Hiệu ứng chuyển đổi |
| `currentSlide` | `number` | `0` | Current active slide index <br/> Chỉ số slide đang hoạt động |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `afterChange` | `number` | Emitted after slide changes <br/> Phát ra sau khi slide thay đổi |
| `beforeChange` | `number` | Emitted before slide changes <br/> Phát ra trước khi slide thay đổi |

### CarouselItemComponent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| Content | `any` | - | Slide content via ng-content <br/> Nội dung slide qua ng-content |

## Accessibility | Khả Năng Tiếp Cận

- **Keyboard navigation** | **Điều hướng bàn phím**: Left/right arrows for slide control
- **ARIA attributes** | **Thuộc tính ARIA**: Proper role="region" and aria-label
- **Focus management** | **Quản lý focus**: Focus visible on navigation elements
- **Pause on hover** | **Tạm dừng khi hover**: Autoplay pauses on mouse hover
- **Alt text** | **Văn bản thay thế**: Ensure images have descriptive alt text

## Best Practices | Thực Hành Tốt Nhất

1. **Image optimization** | **Tối ưu ảnh**: Use optimized images to ensure fast loading

2. **Slide count** | **Số lượng slide**: Keep between 3-7 slides for best UX

3. **Autoplay speed** | **Tốc độ tự động phát**: 3-6 seconds is ideal, adjust based on content

4. **Pause on interaction** | **Tạm dừng khi tương tác**: Autoplay pauses on hover/focus

5. **Mobile responsiveness** | **Đáp ứng di động**: Ensure slides look good on all screen sizes

6. **Alt text** | **Văn bản thay thế**: Always provide descriptive alt text for images

7. **Loading states** | **Trạng thái tải**: Show skeleton or spinner while images load

8. **Content length** | **Độ dài nội dung**: Keep text concise, especially on slides
