The **Badge** component provides a versatile way to display counts, notifications, status indicators, and ribbons.

**Badge** là component linh hoạt để hiển thị số đếm, thông báo, chỉ thị trạng thái và ruy băng.

## Features | Tính Năng

- **Multiple modes** | **Nhiều chế độ**: Dot, count, text, and status display
- **Status types** | **Loại trạng thái**: 6 status colors (success, error, warning, info, default, processing)
- **Processing animation** | **Hiệu ứng xử lý**: Animated dot for processing state
- **Ribbon variant** | **Biến thể ruy băng**: Corner ribbon for cards and containers
- **Overflow count** | **Số tràn**: Show "99+" for large numbers
- **Custom positioning** | **Vị trí tùy chỉnh**: Offset control for badge placement
- **Dark mode** | **Chế độ tối**: Full dark mode support

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component } from '@angular/core';
import { BadgeComponent } from '@/components/ui/badge';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [BadgeComponent],
  template: `
    <!-- Count badge | Badge số đếm -->
    <g-badge [count]="5">
      <button>Messages</button>
    </g-badge>

    <!-- Dot badge | Badge chấm -->
    <g-badge [dot]="true">
      <button>Notifications</button>
    </g-badge>

    <!-- Status badge | Badge trạng thái -->
    <g-badge status="success" text="Online"></g-badge>
  `
})
export class ExampleComponent {}
```

### Count Badge | Badge Số Đếm

Display numerical counts | Hiển thị số đếm:

```html
<!-- Basic count | Số đếm cơ bản -->
<g-badge [count]="5">
  <button class="px-4 py-2 bg-gray-100 rounded">
    Messages
  </button>
</g-badge>

<!-- Large count with overflow | Số đếm lớn với tràn -->
<g-badge [count]="100" [overflowCount]="99">
  <button class="px-4 py-2 bg-gray-100 rounded">
    Notifications
  </button>
</g-badge>
<!-- Shows "99+" -->

<!-- Custom overflow | Tràn tùy chỉnh -->
<g-badge [count]="1000" [overflowCount]="999">
  <button class="px-4 py-2 bg-gray-100 rounded">
    Items
  </button>
</g-badge>
<!-- Shows "999+" -->

<!-- Show zero | Hiển thị số không -->
<g-badge [count]="0" [showZero]="true">
  <button class="px-4 py-2 bg-gray-100 rounded">
    Cart
  </button>
</g-badge>
```

### Custom Positioning | Vị Trí Tùy Chỉnh

Adjust badge position | Điều chỉnh vị trí badge:

```html
<!-- Default position (top-right) | Vị trí mặc định (góc phải trên) -->
<g-badge [count]="5">
  <button>Default</button>
</g-badge>

<!-- Custom offset [x, y] | Offset tùy chỉnh [x, y] -->
<g-badge [count]="5" [offset]="[10, 0]">
  <button>Offset Right</button>
</g-badge>

<g-badge [count]="5" [offset]="[0, 10]">
  <button>Offset Down</button>
</g-badge>

<g-badge [count]="5" [offset]="[-10, -10]">
  <button>Offset Custom</button>
</g-badge>
```

## Props | Thuộc Tính

### BadgeComponent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `count` | `number` | - | Count to display <br/> Số đếm để hiển thị |
| `dot` | `boolean` | `false` | Show as dot <br/> Hiển thị dạng chấm |
| `text` | `string` | - | Text to display <br/> Văn bản để hiển thị |
| `status` | `'success' \| 'error' \| 'warning' \| 'info' \| 'default' \| 'processing'` | - | Status type <br/> Loại trạng thái |
| `overflowCount` | `number` | `99` | Max count before showing + <br/> Số đếm tối đa trước khi hiển thị + |
| `showZero` | `boolean` | `false` | Show badge when count is 0 <br/> Hiển thị badge khi số đếm là 0 |
| `offset` | `[number, number]` | - | Badge position offset [x, y] <br/> Offset vị trí badge [x, y] |
| `size` | `'small' \| 'medium'` | `'medium'` | Badge size <br/> Kích thước badge |
| `color` | `string` | - | Custom background color <br/> Màu nền tùy chỉnh |

### BadgeRibbonComponent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `text` | `string` | required | Ribbon text <br/> Văn bản ruy băng |
| `color` | `string` | - | Custom background color <br/> Màu nền tùy chỉnh |
| `placement` | `'start' \| 'end'` | `'end'` | Corner placement <br/> Vị trí góc |

## Best Practices | Thực Hành Tốt Nhất

1. **Use appropriate modes** | **Sử dụng chế độ phù hợp**:
   - Dot for simple indicators | Chấm cho chỉ thị đơn giản
   - Count for notifications | Số đếm cho thông báo
   - Text for labels | Chữ cho nhãn
   - Status for user states | Trạng thái cho trạng thái người dùng

2. **Set overflow count** | **Đặt số tràn**: Use appropriate `overflowCount` (99 for notifications, 999 for items)

3. **Show zero when needed** | **Hiển thị số không khi cần**: Set `showZero={true}` for shopping carts to show empty state

4. **Color coding** | **Mã màu**: Use consistent colors across your app for status meanings

5. **Processing state** | **Trạng thái xử lý**: Use `status="processing"` for loading states

6. **Ribbon placement** | **Vị trí ruy băng**: Use ribbons sparingly to avoid visual clutter
