The **Splitter** component provides a resizable split-pane layout, perfect for creating adjustable sidebars and multi-panel interfaces.

## Features

- Horizontal and vertical orientations
- Resizable panes by dragging
- Min/max size constraints
- Configurable gutter (divider) size
- Smooth resize interactions
- Keyboard accessible

## Installation

```bash
npx galaxy-ui-cli@latest add splitter
```

## Usage

### Basic Horizontal Splitter

```typescript
import { Component } from '@angular/core';
import { SplitterComponent, SplitterPaneComponent } from '@/components/ui/splitter';

@Component({
  standalone: true,
  imports: [SplitterComponent, SplitterPaneComponent],
  template: `
    <galaxy-splitter [splitterOrientation]="'horizontal'" style="height: 500px;">
      <galaxy-splitter-pane [paneSize]="30">
        <div class="p-4">
          <h3>Sidebar</h3>
          <p>30% width</p>
        </div>
      </galaxy-splitter-pane>

      <galaxy-splitter-pane [paneSize]="70">
        <div class="p-4">
          <h3>Main Content</h3>
          <p>70% width</p>
        </div>
      </galaxy-splitter-pane>
    </galaxy-splitter>
  `
})
```

### Vertical Splitter

```html
<galaxy-splitter [splitterOrientation]="'vertical'" style="height: 600px;">
  <galaxy-splitter-pane [paneSize]="60">
    <div class="p-4">Top Panel</div>
  </galaxy-splitter-pane>

  <galaxy-splitter-pane [paneSize]="40">
    <div class="p-4">Bottom Panel</div>
  </galaxy-splitter-pane>
</galaxy-splitter>
```

### With Size Constraints

```html
<galaxy-splitter style="height: 500px;">
  <galaxy-splitter-pane
    [paneSize]="25"
    [paneMinSize]="15"
    [paneMaxSize]="40">
    Sidebar (min 15%, max 40%)
  </galaxy-splitter-pane>

  <galaxy-splitter-pane [paneSize]="75">
    Main content
  </galaxy-splitter-pane>
</galaxy-splitter>
```

### Non-Resizable Pane

```html
<galaxy-splitter>
  <galaxy-splitter-pane
    [paneSize]="20"
    [paneResizable]="false">
    Fixed sidebar
  </galaxy-splitter-pane>

  <galaxy-splitter-pane [paneSize]="80">
    Resizable content
  </galaxy-splitter-pane>
</galaxy-splitter>
```

### Custom Gutter Size

```html
<galaxy-splitter [splitterGutterSize]="12">
  <galaxy-splitter-pane [paneSize]="50">Left</galaxy-splitter-pane>
  <galaxy-splitter-pane [paneSize]="50">Right</galaxy-splitter-pane>
</galaxy-splitter>
```

## Props

### Splitter

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `splitterOrientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Split direction |
| `splitterGutterSize` | `number` | `8` | Divider size in pixels |

### Splitter Pane

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `paneSize` | `number` | `50` | Initial size (percentage) |
| `paneMinSize` | `number` | `10` | Minimum size (percentage) |
| `paneMaxSize` | `number` | `90` | Maximum size (percentage) |
| `paneResizable` | `boolean` | `true` | Allow resizing |


## Notes

- Always set a height on the splitter container
- Sizes are in percentages and should add up to 100%
- The gutter (resize handle) appears between panes
- Drag the gutter to resize adjacent panes
- Use constraints to prevent panes from becoming too small
