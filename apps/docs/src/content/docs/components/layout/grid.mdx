title: Grid

---

The **Grid** component provides a flexible CSS Grid-based layout system with responsive breakpoints.

## Features

- 12-column grid system
- Responsive breakpoints (sm, md, lg, xl)
- Column and row spanning
- Flexible gap sizes
- Alignment control

## Installation

```bash
npx galaxy-ui-cli@latest add grid
```

## Usage

### Basic Grid

```typescript
import { Component } from '@angular/core';
import { GridComponent, GridColComponent } from '@/components/ui/grid';

@Component({
  standalone: true,
  imports: [GridComponent, GridColComponent],
  template: `
    <galaxy-grid [gridCols]="3" [gridGap]="'md'">
      <galaxy-grid-col>Column 1</galaxy-grid-col>
      <galaxy-grid-col>Column 2</galaxy-grid-col>
      <galaxy-grid-col>Column 3</galaxy-grid-col>
    </galaxy-grid>
  `
})
```

### Column Spanning

```html
<galaxy-grid [gridCols]="12" [gridGap]="'lg'">
  <galaxy-grid-col [colSpan]="8">
    Main content (8 columns)
  </galaxy-grid-col>
  <galaxy-grid-col [colSpan]="4">
    Sidebar (4 columns)
  </galaxy-grid-col>
</galaxy-grid>
```

### Responsive Grid

```html
<galaxy-grid
  [gridCols]="1"
  [gridColsSm]="2"
  [gridColsMd]="3"
  [gridColsLg]="4"
  [gridGap]="'md'">
  <galaxy-grid-col>Item 1</galaxy-grid-col>
  <galaxy-grid-col>Item 2</galaxy-grid-col>
  <galaxy-grid-col>Item 3</galaxy-grid-col>
  <galaxy-grid-col>Item 4</galaxy-grid-col>
</galaxy-grid>
```

### Dashboard Layout

```html
<galaxy-grid [gridCols]="12" [gridGap]="'lg'">
  <!-- Header - Full width -->
  <galaxy-grid-col [colSpan]="12">
    <galaxy-card>
      <galaxy-card-body>Header</galaxy-card-body>
    </galaxy-card>
  </galaxy-grid-col>

  <!-- Sidebar - 3 cols on desktop, full width on mobile -->
  <galaxy-grid-col [colSpan]="12" [colSpanLg]="3">
    <galaxy-card>
      <galaxy-card-body>Sidebar</galaxy-card-body>
    </galaxy-card>
  </galaxy-grid-col>

  <!-- Main - 9 cols on desktop, full width on mobile -->
  <galaxy-grid-col [colSpan]="12" [colSpanLg]="9">
    <galaxy-card>
      <galaxy-card-body>Main Content</galaxy-card-body>
    </galaxy-card>
  </galaxy-grid-col>
</galaxy-grid>
```

## Props

### Grid

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `gridCols` | `GridCols` (1-12) | `12` | Number of columns |
| `gridColsSm` | `GridCols \| null` | `null` | Columns at sm breakpoint |
| `gridColsMd` | `GridCols \| null` | `null` | Columns at md breakpoint |
| `gridColsLg` | `GridCols \| null` | `null` | Columns at lg breakpoint |
| `gridColsXl` | `GridCols \| null` | `null` | Columns at xl breakpoint |
| `gridGap` | `GridGap` | `'md'` | Gap between items |
| `gridAlign` | `GridAlign` | `'stretch'` | Vertical alignment |
| `gridJustify` | `GridJustify` | `'start'` | Horizontal alignment |

### Grid Col

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `colSpan` | `number` | `1` | Number of columns to span |
| `colSpanSm` | `number \| null` | `null` | Span at sm breakpoint |
| `colSpanMd` | `number \| null` | `null` | Span at md breakpoint |
| `colSpanLg` | `number \| null` | `null` | Span at lg breakpoint |
| `colSpanXl` | `number \| null` | `null` | Span at xl breakpoint |
| `colStart` | `number \| null` | `null` | Start column |
| `colEnd` | `number \| null` | `null` | End column |

## Gap Sizes

| Size | Spacing |
|------|---------|
| `none` | 0 |
| `xs` | 0.25rem (4px) |
| `sm` | 0.5rem (8px) |
| `md` | 1rem (16px) |
| `lg` | 1.5rem (24px) |
| `xl` | 2rem (32px) |
| `2xl` | 3rem (48px) |
