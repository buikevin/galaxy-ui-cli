title: Accordion
description: Collapsible content panels for organizing information
---

The **Accordion** component provides collapsible sections for organizing content in a space-efficient way.

## Features

- Single or multiple sections expanded at once
- Status color themes
- Smooth animations
- Disabled items support
- Keyboard accessible

## Installation

```bash
npx galaxy-ui-cli@latest add accordion
```

## Usage

### Basic Accordion

```typescript
import { Component } from '@angular/core';
import { AccordionComponent, AccordionItemComponent } from '@/components/ui/accordion';

@Component({
  standalone: true,
  imports: [AccordionComponent, AccordionItemComponent],
  template: `
    <galaxy-accordion>
      <galaxy-accordion-item [itemTitle]="'What is Galaxy UI?'">
        Galaxy UI is a modern component library for Angular inspired
        by Nebular's design and shadcn/ui's philosophy.
      </galaxy-accordion-item>

      <galaxy-accordion-item [itemTitle]="'How do I install it?'">
        You can install Galaxy UI by running:
        <code>npx galaxy-ui-cli@latest init</code>
      </galaxy-accordion-item>

      <galaxy-accordion-item [itemTitle]="'Is it free?'">
        Yes! Galaxy UI is completely free and open source.
      </galaxy-accordion-item>
    </galaxy-accordion>
  `
})
```

### Multi-Expand Mode

Allow multiple items to be open simultaneously:

```html
<galaxy-accordion [accordionMulti]="true">
  <galaxy-accordion-item [itemTitle]="'Section 1'">
    Content 1
  </galaxy-accordion-item>
  <galaxy-accordion-item [itemTitle]="'Section 2'">
    Content 2
  </galaxy-accordion-item>
</galaxy-accordion>
```

### With Status Colors

```html
<galaxy-accordion [accordionStatus]="'primary'">
  <galaxy-accordion-item [itemTitle]="'Features'" [itemOpen]="true">
    This accordion uses the primary color theme
  </galaxy-accordion-item>
  <galaxy-accordion-item [itemTitle]="'Documentation'">
    More information here
  </galaxy-accordion-item>
</galaxy-accordion>
```

### Disabled Items

```html
<galaxy-accordion>
  <galaxy-accordion-item [itemTitle]="'Available'">
    You can interact with this
  </galaxy-accordion-item>
  <galaxy-accordion-item [itemTitle]="'Disabled'" [itemDisabled]="true">
    This item is disabled
  </galaxy-accordion-item>
</galaxy-accordion>
```

### Default Open State

```html
<galaxy-accordion>
  <galaxy-accordion-item [itemTitle]="'Open by default'" [itemOpen]="true">
    This section is open by default
  </galaxy-accordion-item>
</galaxy-accordion>
```

## Props

### Accordion

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `accordionMulti` | `boolean` | `false` | Allow multiple items open |
| `accordionStatus` | `AccordionStatus` | `'basic'` | Color theme |

### Accordion Item

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `itemTitle` | `string` | `''` | Section title |
| `itemOpen` | `boolean` | `false` | Initial open state |
| `itemDisabled` | `boolean` | `false` | Disable interaction |

## Examples

### FAQ Section

```typescript
@Component({
  template: `
    <div class="max-w-3xl mx-auto">
      <h2 class="text-2xl font-bold mb-6">Frequently Asked Questions</h2>
      <galaxy-accordion [accordionStatus]="'primary'">
        @for (faq of faqs; track faq.id) {
          <galaxy-accordion-item [itemTitle]="faq.question">
            {{ faq.answer }}
          </galaxy-accordion-item>
        }
      </galaxy-accordion>
    </div>
  `
})
export class FaqComponent {
  faqs = [
    { id: 1, question: 'What is...?', answer: '...' },
    { id: 2, question: 'How to...?', answer: '...' },
  ];
}
```

### Settings Accordion

```html
<galaxy-accordion [accordionMulti]="true" [accordionStatus]="'basic'">
  <galaxy-accordion-item [itemTitle]="'Profile Settings'">
    <galaxy-input [inputLabel]="'Display Name'"></galaxy-input>
    <galaxy-input [inputLabel]="'Email'"></galaxy-input>
  </galaxy-accordion-item>

  <galaxy-accordion-item [itemTitle]="'Privacy Settings'">
    <galaxy-checkbox [checkboxLabel]="'Make profile public'"></galaxy-checkbox>
    <galaxy-checkbox [checkboxLabel]="'Show email address'"></galaxy-checkbox>
  </galaxy-accordion-item>

  <galaxy-accordion-item [itemTitle]="'Notifications'">
    <galaxy-checkbox [checkboxLabel]="'Email notifications'"></galaxy-checkbox>
    <galaxy-checkbox [checkboxLabel]="'Push notifications'"></galaxy-checkbox>
  </galaxy-accordion-item>
</galaxy-accordion>
```
