The **Pagination** component provides intuitive controls for navigating through large datasets. Inspired by Ant Design's Pagination component with page size selector, quick jumper, and total display.

**Pagination** là component cung cấp các điều khiển trực quan để điều hướng qua các tập dữ liệu lớn. Lấy cảm hứng từ component Pagination của Ant Design với bộ chọn kích thước trang, bộ nhảy nhanh và hiển thị tổng số.

## Features | Tính Năng

- **Page navigation** | **Điều hướng trang**: Previous, next, and direct page access
- **Page size selector** | **Chọn kích thước trang**: Customizable items per page
- **Quick jumper** | **Nhảy nhanh**: Jump to specific page directly
- **Total display** | **Hiển thị tổng**: Show total items and range
- **Disabled state** | **Trạng thái vô hiệu hóa**: Disable during loading
- **Compact mode** | **Chế độ gọn**: Simplified view for small spaces
- **Dark mode** | **Chế độ tối**: Full dark mode support
- **Responsive** | **Responsive**: Adapts to screen size

## Usage | Sử Dụng

### Basic Usage | Sử Dụng Cơ Bản

```typescript
import { Component, signal } from '@angular/core';
import { PaginationComponent } from '@/components/ui/pagination';

@Component({
  selector: 'app-example',
  standalone: true,
  imports: [PaginationComponent],
  template: `
    <g-pagination
      [total]="total"
      [pageSize]="pageSize()"
      [currentPage]="currentPage()"
      (pageChange)="onPageChange($event)">
    </g-pagination>

    <p>Showing page {{ currentPage() }} of {{ totalPages() }}</p>
  `
})
export class ExampleComponent {
  total = 100;
  pageSize = signal(10);
  currentPage = signal(1);

  totalPages() {
    return Math.ceil(this.total / this.pageSize());
  }

  onPageChange(page: number) {
    this.currentPage.set(page);
    console.log('Load page:', page);
  }
}
```

### With Page Size Selector | Với Chọn Kích Thước Trang

```typescript
@Component({
  template: `
    <g-pagination
      [total]="total"
      [pageSize]="pageSize()"
      [currentPage]="currentPage()"
      [showSizeChanger]="true"
      [pageSizeOptions]="[10, 20, 50, 100]"
      (pageChange)="onPageChange($event)"
      (pageSizeChange)="onPageSizeChange($event)">
    </g-pagination>

    <p>
      Showing {{ getStartIndex() }}-{{ getEndIndex() }} of {{ total }} items
      <br/>
      Hiển thị {{ getStartIndex() }}-{{ getEndIndex() }} trong {{ total }} mục
    </p>
  `
})
export class PageSizeSelectorComponent {
  total = 250;
  pageSize = signal(20);
  currentPage = signal(1);

  getStartIndex() {
    return (this.currentPage() - 1) * this.pageSize() + 1;
  }

  getEndIndex() {
    return Math.min(this.currentPage() * this.pageSize(), this.total);
  }

  onPageChange(page: number) {
    this.currentPage.set(page);
  }

  onPageSizeChange(size: number) {
    this.pageSize.set(size);
    this.currentPage.set(1); // Reset to first page
  }
}
```

### With Quick Jumper | Với Nhảy Nhanh

```typescript
@Component({
  template: `
    <div class="space-y-4">
      <div class="text-lg font-semibold">
        Search Results | Kết Quả Tìm Kiếm: {{ total }} items
      </div>

      <!-- Your content here -->
      <div class="grid grid-cols-3 gap-4">
        @for (item of currentItems(); track item.id) {
          <div class="border rounded p-4">{{ item.title }}</div>
        }
      </div>

      <g-pagination
        [total]="total"
        [pageSize]="pageSize()"
        [currentPage]="currentPage()"
        [showQuickJumper]="true"
        [showTotal]="true"
        (pageChange)="onPageChange($event)">
      </g-pagination>
    </div>
  `
})
export class QuickJumperComponent {
  total = 500;
  pageSize = signal(15);
  currentPage = signal(1);

  // Mock data
  allItems = Array.from({ length: 500 }, (_, i) => ({
    id: i + 1,
    title: `Item ${i + 1}`
  }));

  currentItems = signal(this.allItems.slice(0, 15));

  onPageChange(page: number) {
    this.currentPage.set(page);
    const start = (page - 1) * this.pageSize();
    const end = start + this.pageSize();
    this.currentItems.set(this.allItems.slice(start, end));
  }
}
```

### Disabled State | Trạng Thái Vô Hiệu Hóa

```typescript
@Component({
  template: `
    <div class="space-y-4">
      <g-pagination
        [total]="total"
        [pageSize]="pageSize"
        [currentPage]="currentPage()"
        [disabled]="loading()"
        (pageChange)="loadPage($event)">
      </g-pagination>

      @if (loading()) {
        <div class="text-center">
          <div class="spinner"></div>
          <p>Loading... | Đang tải...</p>
        </div>
      }
    </div>
  `
})
export class DisabledStateComponent {
  total = 200;
  pageSize = 20;
  currentPage = signal(1);
  loading = signal(false);

  async loadPage(page: number) {
    this.loading.set(true);
    this.currentPage.set(page);

    try {
      // Simulate API call
      await new Promise(resolve => setTimeout(resolve, 1000));
      console.log('Loaded page:', page);
    } finally {
      this.loading.set(false);
    }
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `total` | `number` | `0` | Total number of items <br/> Tổng số mục |
| `pageSize` | `number` | `10` | Items per page <br/> Số mục mỗi trang |
| `currentPage` | `number` | `1` | Current page number <br/> Số trang hiện tại |
| `showSizeChanger` | `boolean` | `false` | Show page size selector <br/> Hiển thị bộ chọn kích thước trang |
| `pageSizeOptions` | `number[]` | `[10, 20, 50, 100]` | Page size options <br/> Tùy chọn kích thước trang |
| `showQuickJumper` | `boolean` | `false` | Show quick page jumper <br/> Hiển thị bộ nhảy trang nhanh |
| `showTotal` | `boolean` | `false` | Show total items <br/> Hiển thị tổng số mục |
| `totalTemplate` | `TemplateRef` | - | Custom total display template <br/> Template tùy chỉnh hiển thị tổng |
| `disabled` | `boolean` | `false` | Disable pagination <br/> Vô hiệu hóa phân trang |
| `compact` | `boolean` | `false` | Compact mode <br/> Chế độ gọn |
| `simple` | `boolean` | `false` | Simple mode (arrows only) <br/> Chế độ đơn giản (chỉ mũi tên) |
| `maxPages` | `number` | `7` | Max page buttons to show <br/> Số nút trang tối đa hiển thị |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `pageChange` | `number` | Emitted when page changes <br/> Phát ra khi trang thay đổi |
| `pageSizeChange` | `number` | Emitted when page size changes <br/> Phát ra khi kích thước trang thay đổi |

## Best Practices | Thực Hành Tốt Nhất

1. **Show total count** | **Hiển thị tổng số**: Help users understand the data size

2. **Preserve page state** | **Giữ trạng thái trang**: Maintain page number in URL or state

3. **Disable during loading** | **Vô hiệu hóa khi tải**: Prevent multiple requests

4. **Reasonable page sizes** | **Kích thước trang hợp lý**: Offer appropriate options (10, 20, 50, 100)

5. **Scroll to top** | **Cuộn lên đầu**: Auto-scroll when page changes

6. **Quick jumper for large datasets** | **Nhảy nhanh cho dữ liệu lớn**: Enable for 10+ pages
