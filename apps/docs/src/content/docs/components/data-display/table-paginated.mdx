The **Table Paginated** component combines the power of the Table component with built-in pagination, providing a complete solution for displaying large datasets. All features from both components work seamlessly together.

**Table Paginated** là component kết hợp sức mạnh của component Table với phân trang tích hợp sẵn, cung cấp giải pháp hoàn chỉnh để hiển thị các tập dữ liệu lớn. Tất cả tính năng từ cả hai component hoạt động liền mạch cùng nhau.

## Features | Tính Năng

- **All table features** | **Tất cả tính năng bảng**: Sorting, selection, expandable rows, custom rendering
- **Built-in pagination** | **Phân trang tích hợp**: Seamless page navigation controls
- **Server-side ready** | **Sẵn sàng server-side**: Supports both client and server pagination
- **Page size control** | **Điều khiển kích thước trang**: Dynamic items per page
- **Quick jumper** | **Nhảy nhanh**: Direct page navigation
- **Loading states** | **Trạng thái tải**: Integrated loading indicators
- **Dark mode** | **Chế độ tối**: Full dark mode support
- **Responsive** | **Responsive**: Adapts to all screen sizes

### Client-Side Pagination | Phân Trang Phía Client

```typescript
@Component({
  template: `
    <g-table-paginated
      [columns]="columns"
      [data]="currentPageData()"
      [total]="allData.length"
      [pageSize]="pageSize()"
      [currentPage]="currentPage()"
      [showSizeChanger]="true"
      [pageSizeOptions]="[10, 20, 50]"
      (pageChange)="onPageChange($event)"
      (pageSizeChange)="onPageSizeChange($event)">
    </g-table-paginated>
  `
})
export class ClientSidePaginationComponent {
  pageSize = signal(10);
  currentPage = signal(1);

  columns = [
    { key: 'name', label: 'Product | Sản phẩm', sortable: true },
    { key: 'category', label: 'Category | Danh mục', sortable: true },
    { key: 'price', label: 'Price | Giá', sortable: true },
    { key: 'stock', label: 'Stock | Kho', sortable: true }
  ];

  // All data loaded at once
  allData = Array.from({ length: 500 }, (_, i) => ({
    id: i + 1,
    name: `Product ${i + 1}`,
    category: ['Electronics', 'Clothing', 'Food', 'Books'][i % 4],
    price: Math.random() * 1000,
    stock: Math.floor(Math.random() * 100)
  }));

  currentPageData = signal(this.allData.slice(0, 10));

  onPageChange(page: number) {
    this.currentPage.set(page);
    this.updateCurrentPageData();
  }

  onPageSizeChange(size: number) {
    this.pageSize.set(size);
    this.currentPage.set(1);
    this.updateCurrentPageData();
  }

  updateCurrentPageData() {
    const start = (this.currentPage() - 1) * this.pageSize();
    const end = start + this.pageSize();
    this.currentPageData.set(this.allData.slice(start, end));
  }
}
```

### Server-Side Pagination | Phân Trang Phía Server

```typescript
import { HttpClient } from '@angular/common/http';

@Component({
  template: `
    <g-table-paginated
      [columns]="columns"
      [data]="data()"
      [total]="total()"
      [pageSize]="pageSize()"
      [currentPage]="currentPage()"
      [loading]="loading()"
      [sortable]="true"
      [showSizeChanger]="true"
      [showQuickJumper]="true"
      (pageChange)="onPageChange($event)"
      (pageSizeChange)="onPageSizeChange($event)"
      (sortChange)="onSortChange($event)">
    </g-table-paginated>
  `
})
export class ServerSidePaginationComponent {
  private http = inject(HttpClient);

  loading = signal(false);
  data = signal<any[]>([]);
  total = signal(0);
  pageSize = signal(20);
  currentPage = signal(1);
  sortColumn = signal('');
  sortDirection = signal<'asc' | 'desc'>('asc');

  columns = [
    { key: 'name', label: 'User | Người dùng', sortable: true },
    { key: 'email', label: 'Email', sortable: true },
    { key: 'role', label: 'Role | Vai trò', sortable: true },
    { key: 'status', label: 'Status | Trạng thái', sortable: true },
    { key: 'createdAt', label: 'Created | Tạo lúc', sortable: true }
  ];

  ngOnInit() {
    this.loadData();
  }

  async loadData() {
    this.loading.set(true);

    try {
      const params = {
        page: this.currentPage(),
        pageSize: this.pageSize(),
        sortBy: this.sortColumn(),
        sortDirection: this.sortDirection()
      };

      const response = await this.http.get<any>('/api/users', { params }).toPromise();

      this.data.set(response.data);
      this.total.set(response.total);
    } catch (error) {
      console.error('Failed to load data:', error);
    } finally {
      this.loading.set(false);
    }
  }

  onPageChange(page: number) {
    this.currentPage.set(page);
    this.loadData();
  }

  onPageSizeChange(size: number) {
    this.pageSize.set(size);
    this.currentPage.set(1);
    this.loadData();
  }

  onSortChange(event: { column: string; direction: 'asc' | 'desc' }) {
    this.sortColumn.set(event.column);
    this.sortDirection.set(event.direction);
    this.currentPage.set(1);
    this.loadData();
  }
}
```

## Props | Thuộc Tính

Combines all props from both Table and Pagination components:

### Table Props | Thuộc Tính Bảng

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `columns` | `TableColumn[]` | `[]` | Column definitions <br/> Định nghĩa cột |
| `data` | `any[]` | `[]` | Current page data <br/> Dữ liệu trang hiện tại |
| `selectable` | `boolean` | `false` | Enable row selection <br/> Bật chọn hàng |
| `selectedRows` | `any[]` | `[]` | Selected rows <br/> Các hàng đã chọn |
| `expandable` | `boolean` | `false` | Enable expandable rows <br/> Bật hàng mở rộng |
| `sortable` | `boolean` | `false` | Enable sorting <br/> Bật sắp xếp |
| `loading` | `boolean` | `false` | Show loading state <br/> Hiển thị trạng thái tải |
| `rowKey` | `string` | `'id'` | Unique row identifier <br/> Định danh duy nhất |

### Pagination Props | Thuộc Tính Phân Trang

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `total` | `number` | `0` | Total items across all pages <br/> Tổng số mục trên tất cả trang |
| `pageSize` | `number` | `10` | Items per page <br/> Số mục mỗi trang |
| `currentPage` | `number` | `1` | Current page number <br/> Số trang hiện tại |
| `showSizeChanger` | `boolean` | `false` | Show page size selector <br/> Hiển thị bộ chọn kích thước |
| `pageSizeOptions` | `number[]` | `[10, 20, 50, 100]` | Page size options <br/> Tùy chọn kích thước trang |
| `showQuickJumper` | `boolean` | `false` | Show quick jumper <br/> Hiển thị bộ nhảy nhanh |
| `showTotal` | `boolean` | `false` | Show total count <br/> Hiển thị tổng số |

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `pageChange` | `number` | Page changed <br/> Trang đã thay đổi |
| `pageSizeChange` | `number` | Page size changed <br/> Kích thước trang đã thay đổi |
| `selectionChange` | `any[]` | Selection changed <br/> Lựa chọn đã thay đổi |
| `sortChange` | `{ column: string; direction: 'asc' \| 'desc' }` | Sort changed <br/> Sắp xếp đã thay đổi |
| `rowClick` | `any` | Row clicked <br/> Hàng được nhấp |

## Best Practices | Thực Hành Tốt Nhất

1. **Server-side for large data** | **Server-side cho dữ liệu lớn**: Use server pagination for 1000+ items

2. **Clear selection on page change** | **Xóa lựa chọn khi đổi trang**: Reset selection when navigating

3. **Loading indicators** | **Chỉ báo tải**: Show loading state during data fetch

4. **Preserve filters** | **Giữ bộ lọc**: Maintain filters across page changes

5. **Responsive page sizes** | **Kích thước trang responsive**: Adjust default page size for mobile

6. **URL synchronization** | **Đồng bộ URL**: Keep page state in URL for bookmarking
