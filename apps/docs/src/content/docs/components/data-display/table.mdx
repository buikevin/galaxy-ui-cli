import { Tabs, TabItem } from '@astrojs/starlight/components';

The **Table** component provides a powerful and flexible way to display tabular data with built-in support for sorting, row selection, expandable rows, and custom rendering. Inspired by Ant Design's Table component.

**Table** là component cung cấp cách mạnh mẽ và linh hoạt để hiển thị dữ liệu dạng bảng với hỗ trợ sẵn có cho sắp xếp, chọn hàng, hàng mở rộng và tùy chỉnh hiển thị. Lấy cảm hứng từ component Table của Ant Design.

## Features | Tính Năng

- **Column sorting** | **Sắp xếp cột**: Sort data by any column with visual indicators
- **Row selection** | **Chọn hàng**: Single or multiple row selection with checkboxes
- **Expandable rows** | **Hàng mở rộng**: Expand rows to show additional details
- **Custom rendering** | **Tùy chỉnh hiển thị**: Custom cell and header templates
- **Pagination ready** | **Sẵn sàng phân trang**: Works seamlessly with pagination component
- **Responsive design** | **Thiết kế responsive**: Horizontal scroll on small screens
- **Dark mode** | **Chế độ tối**: Full dark mode support
- **Loading state** | **Trạng thái tải**: Built-in loading skeleton

  ]);

  onSortChange(event: { column: string; direction: 'asc' | 'desc' }) {
    this.currentSort.set(event);
    // Sort data logic here
  }
}

### With Custom Cell Rendering | Với Tùy Chỉnh Ô

```typescript
@Component({
  template: `
    <g-table
      [columns]="columns"
      [data]="data()">
      <ng-template #cellTemplate let-column="column" let-row="row">
        @if (column.key === 'status') {
          <span [class]="getStatusClass(row.status)">
            {{ row.status }}
          </span>
        } @else if (column.key === 'actions') {
          <div class="flex gap-2">
            <button (click)="edit(row)" class="btn-sm btn-primary">
              Edit | Sửa
            </button>
            <button (click)="delete(row)" class="btn-sm btn-danger">
              Delete | Xóa
            </button>
          </div>
        } @else {
          {{ row[column.key] }}
        }
      </ng-template>
    </g-table>
  `
})
export class CustomCellTableComponent {
  columns = [
    { key: 'name', label: 'Name | Tên' },
    { key: 'status', label: 'Status | Trạng thái' },
    { key: 'actions', label: 'Actions | Thao tác' }

  data = signal([
    { id: 1, name: 'Task 1', status: 'completed' },
    { id: 2, name: 'Task 2', status: 'pending' },

      'completed': 'px-2 py-1 rounded bg-green-100 text-green-800 dark:bg-green-900 dark:text-green-200',

      'in-progress': 'px-2 py-1 rounded bg-blue-100 text-blue-800 dark:bg-blue-900 dark:text-blue-200'
    };
    return classes[status] || '';
  }

  edit(row: any) {
    console.log('Edit:', row);
  }

  delete(row: any) {
    console.log('Delete:', row);
  }
}
```

## Props | Thuộc Tính

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `columns` | `TableColumn[]` | `[]` | Column definitions <br/> Định nghĩa cột |
| `data` | `any[]` | `[]` | Table data array <br/> Mảng dữ liệu bảng |
| `selectable` | `boolean` | `false` | Enable row selection <br/> Bật chọn hàng |
| `selectedRows` | `any[]` | `[]` | Currently selected rows <br/> Các hàng đang được chọn |
| `expandable` | `boolean` | `false` | Enable expandable rows <br/> Bật hàng mở rộng |
| `sortable` | `boolean` | `false` | Enable column sorting <br/> Bật sắp xếp cột |
| `loading` | `boolean` | `false` | Show loading state <br/> Hiển thị trạng thái tải |
| `emptyText` | `string` | `'No data'` | Text when no data <br/> Văn bản khi không có dữ liệu |
| `rowKey` | `string` | `'id'` | Unique row identifier <br/> Định danh duy nhất cho hàng |

### TableColumn Interface

```typescript
interface TableColumn {
  key: string;           // Data key | Khóa dữ liệu
  label: string;         // Column header | Tiêu đề cột
  sortable?: boolean;    // Enable sorting | Bật sắp xếp
  width?: string;        // Column width | Độ rộng cột
  align?: 'left' | 'center' | 'right'; // Text alignment | Căn chỉnh
  render?: (value: any, row: any) => any; // Custom renderer | Tùy chỉnh hiển thị
}
```

### Outputs | Sự Kiện

| Event | Type | Description |
|-------|------|-------------|
| `selectionChange` | `any[]` | Emitted when selection changes <br/> Phát ra khi lựa chọn thay đổi |
| `sortChange` | `{ column: string; direction: 'asc' \| 'desc' }` | Emitted when sort changes <br/> Phát ra khi sắp xếp thay đổi |
| `rowClick` | `any` | Emitted when row is clicked <br/> Phát ra khi hàng được nhấp |
| `expandChange` | `{ row: any; expanded: boolean }` | Emitted when row expand state changes <br/> Phát ra khi trạng thái mở rộng thay đổi |

## Best Practices | Thực Hành Tốt Nhất

1. **Use consistent column keys** | **Dùng khóa cột nhất quán**: Match data keys exactly with column keys

2. **Provide loading states** | **Cung cấp trạng thái tải**: Show skeleton or spinner while fetching data

3. **Limit initial data** | **Giới hạn dữ liệu ban đầu**: Use pagination for large datasets

4. **Custom cell rendering** | **Tùy chỉnh ô**: Use templates for complex cell content

5. **Responsive design** | **Thiết kế responsive**: Consider horizontal scroll or card view on mobile

6. **Accessible labels** | **Nhãn dễ tiếp cận**: Provide clear column headers and ARIA labels
