The **Menu** component provides a flexible navigation menu system with support for icons, badges, groups, and submenu items.

## Features

- Lucide icons integration
- Active state highlighting
- Badge notifications
- Grouped menu items
- Nested submenu support
- Compact mode
- Status color themes
- Keyboard accessible

## Installation

```bash
npx galaxy-ui-cli@latest add menu
```

## Usage

### Basic Menu

```typescript
import { Component } from '@angular/core';
import { MenuComponent, MenuItemComponent } from '@/components/ui/menu';
import { LucideAngularModule } from 'lucide-angular';

@Component({
  standalone: true,
  imports: [MenuComponent, MenuItemComponent, LucideAngularModule],
  template: `
    <g-menu>
      <g-menu-item
        [itemIcon]="'home'"
        [itemLabel]="'Home'"
        [itemActive]="true">
      </g-menu-item>
      <g-menu-item
        [itemIcon]="'inbox'"
        [itemLabel]="'Messages'">
      </g-menu-item>
      <g-menu-item
        [itemIcon]="'settings'"
        [itemLabel]="'Settings'">
      </g-menu-item>
    </g-menu>
  `
})
```

## Props

### Menu

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `menuStatus` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'basic'` | Color theme |
| `menuCompact` | `boolean` | `false` | Reduce spacing |

### Menu Item

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `itemIcon` | `string \| null` | `null` | Lucide icon name |
| `itemLabel` | `string` | `''` | Item text |
| `itemActive` | `boolean` | `false` | Active state |
| `itemDisabled` | `boolean` | `false` | Disabled state |
| `itemBadge` | `string \| number \| null` | `null` | Badge content |
| `itemHasSubmenu` | `boolean` | `false` | Show submenu indicator |
| `itemExpanded` | `boolean` | `false` | Submenu expanded |

### Menu Group

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `groupTitle` | `string \| null` | `null` | Group heading text |
| `groupCollapsible` | `boolean` | `false` | Allow collapse |
| `groupExpanded` | `boolean` | `true` | Initial state |

## Best Practices

1. **Use semantic icons** - Match icons to menu item purpose
2. **Limit nesting** - Keep submenu depth to 1-2 levels
3. **Highlight active** - Always show current page
4. **Badge sparingly** - Only for important notifications
5. **Group logically** - Related items together

## Accessibility

- Use semantic HTML for menu structure
- Provide `aria-label` for icon-only items
- Support keyboard navigation
- Maintain focus indicators
- Use proper heading hierarchy for groups
