The **Sidebar** component provides a flexible sidebar layout with header, content, and footer sections.

## Features

- Left or right positioning
- Fixed or overlay modes
- Collapsible with toggle button
- Custom width (expanded and collapsed)
- Optional backdrop for overlay mode
- Status color themes
- Responsive (auto-collapse on mobile)
- Keyboard support (Escape to close)

## Installation

```bash
npx galaxy-ui-cli@latest add sidebar
```

### Collapsible Sidebar

```typescript
@Component({
  template: `
    <div class="flex h-screen">
      <g-sidebar
        [sidebarCollapsed]="isCollapsed()"
        (sidebarCollapsedChange)="onCollapseChange($event)">
        <g-sidebar-header>
          <div class="flex items-center gap-2">
            <lucide-angular [img]="'zap'" class="w-6 h-6 text-violet-600"></lucide-angular>
            @if (!isCollapsed()) {
              <span class="font-bold">Galaxy UI</span>
            }
          </div>
        </g-sidebar-header>

        <g-sidebar-content>
          <g-menu [menuCompact]="isCollapsed()">
            <g-menu-item
              [itemIcon]="'home'"
              [itemLabel]="'Dashboard'"
              [itemActive]="true">
            </g-menu-item>
            <g-menu-item
              [itemIcon]="'users'"
              [itemLabel]="'Team'">
            </g-menu-item>
            <g-menu-item
              [itemIcon]="'settings'"
              [itemLabel]="'Settings'">
            </g-menu-item>
          </g-menu>
        </g-sidebar-content>
      </g-sidebar>

      <main class="flex-1 overflow-auto p-8">
        <h1>Main Content</h1>
      </main>
    </div>
  `
})
export class CollapsibleSidebarExample {
  isCollapsed = signal(false);

  onCollapseChange(collapsed: boolean) {
    this.isCollapsed.set(collapsed);
  }
}
```

### Overlay Mode


## Props

### Sidebar

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `sidebarPosition` | `'left' \| 'right'` | `'left'` | Sidebar position |
| `sidebarMode` | `'fixed' \| 'overlay'` | `'fixed'` | Layout mode |
| `sidebarStatus` | `'basic' \| 'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | `'basic'` | Color theme |
| `sidebarCollapsed` | `boolean` | `false` | Collapsed state |
| `sidebarCollapsible` | `boolean` | `true` | Show collapse button |
| `sidebarWidth` | `string` | `'16rem'` | Expanded width |
| `sidebarCollapsedWidth` | `string` | `'4rem'` | Collapsed width |
| `sidebarBackdrop` | `boolean` | `true` | Show backdrop (overlay mode) |
| `sidebarResponsive` | `boolean` | `true` | Auto-collapse on mobile |

### Events

| Event | Type | Description |
|-------|------|-------------|
| `sidebarCollapsedChange` | `EventEmitter<boolean>` | Emits when collapse state changes |

### Mobile Responsive

```typescript
@Component({
  template: `
    <div class="flex h-screen">
      <!-- Mobile: Overlay mode -->
      <g-sidebar
        [sidebarMode]="'overlay'"
        [sidebarCollapsed]="!mobileMenuOpen()"
        (sidebarCollapsedChange)="onMobileMenuToggle($event)"
        class="md:hidden">
        <g-sidebar-content>
          <g-menu>
            <g-menu-item [itemIcon]="'home'" [itemLabel]="'Home'"></g-menu-item>
            <g-menu-item [itemIcon]="'inbox'" [itemLabel]="'Messages'"></g-menu-item>
            <g-menu-item [itemIcon]="'settings'" [itemLabel]="'Settings'"></g-menu-item>
          </g-menu>
        </g-sidebar-content>
      </g-sidebar>

      <!-- Desktop: Fixed mode -->
      <g-sidebar
        [sidebarMode]="'fixed'"
        class="hidden md:block">
        <g-sidebar-content>
          <g-menu>
            <g-menu-item [itemIcon]="'home'" [itemLabel]="'Home'"></g-menu-item>
            <g-menu-item [itemIcon]="'inbox'" [itemLabel]="'Messages'"></g-menu-item>
            <g-menu-item [itemIcon]="'settings'" [itemLabel]="'Settings'"></g-menu-item>
          </g-menu>
        </g-sidebar-content>
      </g-sidebar>

      <!-- Main content -->
      <div class="flex-1 flex flex-col">
        <!-- Mobile header with menu button -->
        <header class="md:hidden p-4 border-b">
          <button gButton (click)="toggleMobileMenu()">
            <lucide-angular [img]="'menu'"></lucide-angular>
          </button>
        </header>

        <main class="flex-1 overflow-auto p-8">
          <h1>Main Content</h1>
        </main>
      </div>
    </div>
  `
})
export class ResponsiveSidebar {
  mobileMenuOpen = signal(false);

  toggleMobileMenu() {
    this.mobileMenuOpen.update(v => !v);
  }

  onMobileMenuToggle(open: boolean) {
    this.mobileMenuOpen.set(!open);
  }
}
```

## Best Practices

1. **Fixed vs Overlay** - Use fixed for desktop, overlay for mobile
2. **Collapse state** - Persist collapsed state in localStorage
3. **Icon-only mode** - Show tooltips when collapsed
4. **Responsive** - Auto-collapse on mobile screens
5. **Z-index** - Ensure sidebar appears above content in overlay mode
6. **Transitions** - Use smooth animations for collapse/expand

## Accessibility

- Support keyboard navigation (Tab, Escape)
- Close overlay on Escape key
- Focus trap when overlay is open
- Provide skip navigation link
- Use semantic HTML (`<nav>`, `<aside>`)
- Label collapse button with `aria-label`

## Keyboard Shortcuts

- **Escape** - Close sidebar (overlay mode)
- **Tab** - Navigate through menu items
