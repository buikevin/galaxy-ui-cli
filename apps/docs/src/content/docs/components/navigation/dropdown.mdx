The **Dropdown** component displays a menu of options when triggered by a button or other element.

## Features

- Click or hover trigger
- Position control (top, bottom, left, right)
- Alignment options (start, center, end)
- Lucide icons support
- Keyboard shortcuts display
- Danger items styling
- Auto-close on selection
- Keyboard navigation (Escape to close)
- Inspired by shadcn/AntD

## Installation

```bash
npx galaxy-ui-cli@latest add dropdown
```

## Usage

### Basic Dropdown

```typescript
import { Component } from '@angular/core';
import {
  DropdownComponent,
  DropdownMenuComponent,
  DropdownItemComponent,
  DropdownTriggerDirective
} from '@/components/ui/dropdown';

@Component({
  standalone: true,
  imports: [
    DropdownComponent,
    DropdownMenuComponent,
    DropdownItemComponent,
    DropdownTriggerDirective
  ],
  template: `
    <g-dropdown>
      <button gButton gDropdownTrigger>
        Actions
      </button>
      <g-dropdown-menu>
        <g-dropdown-item>Edit</g-dropdown-item>
        <g-dropdown-item>Duplicate</g-dropdown-item>
        <g-dropdown-item>Delete</g-dropdown-item>
      </g-dropdown-menu>
    </g-dropdown>
  `
})
```

## Props

### Dropdown

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `dropdownPosition` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Menu position |
| `dropdownAlign` | `'start' \| 'center' \| 'end'` | `'start'` | Menu alignment |
| `dropdownTrigger` | `'click' \| 'hover'` | `'click'` | Trigger method |
| `dropdownOffset` | `number` | `8` | Distance from trigger (px) |

### Dropdown Item

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `itemIcon` | `string \| null` | `null` | Lucide icon name |
| `itemShortcut` | `string \| null` | `null` | Keyboard shortcut text |
| `itemDisabled` | `boolean` | `false` | Disabled state |
| `itemDanger` | `boolean` | `false` | Danger styling (red) |

### Context Menu

```typescript
@Component({
  template: `
    <div
      class="p-8 border-2 border-dashed border-slate-300 dark:border-slate-700 rounded"
      (contextmenu)="openContextMenu($event)">
      Right-click me
    </div>

    <g-dropdown #contextMenu [dropdownTrigger]="'click'">
      <span gDropdownTrigger></span>
      <g-dropdown-menu>
        <g-dropdown-item [itemIcon]="'copy'" [itemShortcut]="'⌘C'">
          Copy
        </g-dropdown-item>
        <g-dropdown-item [itemIcon]="'clipboard'" [itemShortcut]="'⌘V'">
          Paste
        </g-dropdown-item>
        <g-dropdown-item [itemIcon]="'scissors'" [itemShortcut]="'⌘X'">
          Cut
        </g-dropdown-item>

        <g-dropdown-divider></g-dropdown-divider>

        <g-dropdown-item [itemIcon]="'trash-2'" [itemDanger]="true">
          Delete
        </g-dropdown-item>
      </g-dropdown-menu>
    </g-dropdown>
  `
})
export class ContextMenuExample {
  @ViewChild('contextMenu') contextMenu!: DropdownComponent;

  openContextMenu(event: MouseEvent) {
    event.preventDefault();
    this.contextMenu.open();
  }
}
```

## Best Practices

1. **Clear labels** - Use descriptive menu item text
2. **Icons consistently** - Use icons for all items or none
3. **Group related** - Use dividers to separate groups
4. **Destructive actions** - Use `itemDanger` for delete/remove
5. **Keyboard shortcuts** - Show shortcuts when available
6. **Auto-close** - Close menu after item selection
7. **Max items** - Keep menus under 10 items

## Accessibility

- Use semantic button for trigger
- Support keyboard navigation (Tab, Enter, Space, Escape)
- Provide `aria-label` for icon-only triggers
- Close on Escape key
- Focus management when opening/closing
- Use `role="menu"` and `role="menuitem"`

## Keyboard Shortcuts

- **Escape** - Close dropdown
- **Enter/Space** - Select item
- **Tab** - Navigate items
- **Arrow Up/Down** - Navigate items (if implemented)
